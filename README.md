# dazi-work

搭子（Dazi）本地工作区：DRAP 应用项目、Excel 资源、流程与文档。

## 目录

| 路径 | 说明 |
|------|------|
| `项目/` | 应用 / 流程 / 本体等项目 monorepo |
| `资源/` | Excel 设计稿、文档、表结构等 |
| `scripts/` | dazi CLI 脚本（如 `dazi.ps1 app`） |
| `tools/` | 辅助工具 |

## 克隆后

各应用项目目录内执行依赖安装，例如：

```powershell
cd 项目/app_项目应用0520
pnpm install
```
