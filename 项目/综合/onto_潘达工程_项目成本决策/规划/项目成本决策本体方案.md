## 二、需求分析

### 2.1 业务背景

潘达建工是一家中型建筑施工企业，以房屋建筑为主营业务。本项目旨在构建**工程项目成本决策本体**，实现：

- 项目全生命周期成本动态监控与预警
- 业主产值、成本、利润多维度分析
- 资金流预测与回款风险识别
- 经营指标智能诊断与决策建议
- 自然语言交互式成本查询与分析

### 2.2 核心业务问题

| 问题类别         | 具体问题                                   | 决策价值                         |
| ---------------- | ------------------------------------------ | -------------------------------- |
| **成本控制**     | 项目成本是否超支？超支原因是什么？         | 及时纠偏，防止利润流失           |
| **产值管理**     | 业主产值确认进度如何？待确产值有多少？     | 优化产值申报策略，加速回款       |
| **利润分析**     | 项目真实利润是多少？利润率是否达标？       | 准确评估项目盈利能力             |
| **资金风险**     | 项目资金流是否健康？回款风险有多大？       | 预警资金断裂风险，提前应对       |
| **经营诊断**     | 哪些指标异常？如何改进？                   | 智能诊断，给出改进建议           |
| **对比分析**     | 同类项目成本对比？历史项目经验借鉴？       | 横向对标，持续优化               |

### 2.3 业务角色与场景

| 角色           | 核心关注点                           | 典型查询场景                                       |
| -------------- | ------------------------------------ | -------------------------------------------------- |
| **项目经理**   | 本项目成本、产值、利润、资金         | "我这个月成本超支了多少？" "产值确认率怎么样？"    |
| **商务经理**   | 多项目对比、成本预警、回款管理       | "哪些项目利润率低于5%？" "回款风险最大的项目？"   |
| **公司管理层** | 整体经营状况、战略决策               | "本月所有项目利润汇总？" "大客户项目经营情况？"   |
| **财务人员**   | 资金流、应收应付、成本核算           | "本月应收工程款总额？" "资金流预测？"             |

## 三、本体层设计（业务世界优先）

> **设计原则**：先定义业务对象与链接关系，再反推物理表与Cube。对象类型code使用稳定业务英文标识，与表名、Cube名刻意区分。

### 3.3 链接类型（12种业务关系）

> **设计原则**：链接两端必须是对象类型code，表达有向业务关系。

## 十、附录：指标体系

### 10.1 核心经营指标

| 指标编码 | 指标名称               | 计算公式                           | 参考值         |
| -------- | ---------------------- | ---------------------------------- | -------------- |
| [10]     | 业主产值确权比         | 已确权产值 / 总产值                | ≥85%           |
| [20]     | 确权产值回款率         | 业主已收款 / 已确权产值            | ≥75~80%        |
| [30]     | 成本确权比             | 已确权产值 / 总成本                | ≥85%           |
| [40]     | 成本刚性度             | 已确权成本 / 总成本                | ≥95%           |
| [50]     | 产值现金流入比         | 总收入 / 总产值                    | ≥70%           |
| [60]     | 成本资金收入比         | 总收入 / 总成本                    | 根据合同约定   |
| [70]     | 成本资金支出比         | 总支出 / 总成本                    | 65%~80%        |

### 10.2 预警规则

| 预警等级 | 触发条件                                                       |
| -------- | -------------------------------------------------------------- |
| 红色     | 7项指标中≥6项超出参考范围，或总利润率连续3个月为负            |
| 黄色     | 7项指标中2-5项超出参考范围，或总利润率连续2个月为负           |
| 绿色     | 指标正常且利润率为正                                           |
