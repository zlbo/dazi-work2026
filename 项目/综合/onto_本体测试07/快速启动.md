# 快速启动

欢迎开始 **本体项目** 开发。本项目使用 **Python** 编写本体脚本（见 `脚本/` 目录）。

## 1. 本体规划

在 `规划/` 目录中整理对象类型、关系与业务规则。详细方法请参考：

- [本体规划指南](../../资源/docs/onto/本体规划指南.md)
- 同目录可参考：`资源/docs/onto/` 下其他 Onto 文档

## 2. 编写脚本

在 `脚本/` 目录中编写 Python 本体脚本。规范与示例请参考：

- [本体脚本编写指南](../../资源/docs/onto/本体脚本编写指南.md)
- 扩展设置 `dazi.onto.defaultScriptLang` 默认为 `python`

## 3. 数据空间

本项目绑定数据空间：**综合示例数据**（`space__misc_01`）

可在侧栏 **数据资源 → 数据空间** 中预览表结构与样例数据。

## 4. 下一步

1. 阅读项目根目录 `README.md` 了解项目元信息
2. 在 `规划/` 中完成本体设计文档
3. 在 `脚本/` 中实现并调试 Python 脚本
4. 使用侧栏 **Onto 本体** 相关命令发布与运行
