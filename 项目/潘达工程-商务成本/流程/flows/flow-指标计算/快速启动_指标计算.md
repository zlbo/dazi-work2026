## 流程说明

本流程用于基于 flow-001~004 的同步数据计算派生指标（L2-L4指标），输出到 `ProjectIndicator Cube`。

**节点拓扑**:
```
开始 → 获取产值数据 → 获取成本数据 → 获取付款数据 → 获取收支数据
                                           ↓
                                     计算派生指标 → 结束
```

**派生指标计算**:
| 指标名称 | 公式 | 指标层级 |
|---------|------|---------|
| 利润率 | (产值 - 成本) / 产值 | L2 |
| 成本率 | 成本 / 产值 | L2 |
| 回款率 | 已收款 / 应收款 | L2 |
| 毛利率 | (产值 - 完全成本) / 产值 | L3 |
