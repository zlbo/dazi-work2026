## 一、映射关系概述

表3是整个报表体系的**数据核心枢纽**,采用"一行多列"的独特设计。数据流向为:
- **数据来源**: 表3.1、表3.2、项目收支汇总表
- **数据去向**: 表1、表2引用
- **数据库映射**: 通过多个专门表存储数据

| 企业规则ID | 规则名称 | 关联区域 | 说明 |

## 五、映射类型详细说明

### 5.1 直接映射

Excel单元格直接对应数据库字段,通过code关联:

```
Excel:表3!B7
  ↓
code: 'project_name'
  ↓
Database: tb_project_indicator where code='project_name'
  ↓
提取: value字段
```

### 5.2 间接映射(通过子表)

Excel从子表引用,子表再映射到数据库:

```
Excel:表3!V7
  ↓
引用:表3.1!D6
  ↓
子表内部映射到数据库
```

### 5.3 计算映射

通过公式计算得出:

```
X7 = V7 + W7
AG7 = AE7 + AF7
```

### 5.4 LOOKUP映射

通过查找项目收支汇总表:

```
F7 = LOOKUP(1,0/(项目收支汇总表!A5:A306="5.1"),项目收支汇总表!C5:C306)
```
