# 140-表3.1.1项目月度业主待确产值分析表结构分析

## 一、报表基本信息

| 属性 | 值 |
|-----|---|
| 报表名称 | 项目月度业主待确产值分析表 |
| 表编号 | 表3.1.1 |
| Excel Sheet名 | `项目月度业主待确产值分析表-表3.1.1` |
| 工作区维度 | A1:E8 (非常小，仅8行5列) |
| 数据来源 | 手动填写 |
| 填写状态 | 填写(手动录入) |

## 二、报表整体结构

表3.1.1采用**列表型**结构，主要包含4项待确原因的详细说明：

```
┌─────────────────────────────────────────────┐
│ 表头:项目名称、报告期间                      │
├─────────────────────────────────────────────┤
│ A列:序号                                    │
│ B列:项目名称(固定内容)                       │
│ C列:主要内容(手动填写)                       │
│ D列:(空列)                                  │
│ E列:备注                                    │
├─────────────────────────────────────────────┤
│ 第5行: [10]实际形象进度描述                  │
│ 第6行: [20]合同付款摘要                      │
│ 第7行: [30]待确主要原因描述                  │
│ 第8行: [40]待确金额                         │
└─────────────────────────────────────────────┘
```

## 六、业务逻辑分析

### 6.1 待确产值分析流程

```
1. 描述实际形象进度(C5)
    ↓
2. 说明合同付款摘要(C6)
    ↓
3. 分析待确主要原因(C7)
    ↓
4. 填写待确金额(C8)
    ↓
5. 提供附件支撑(E8要求)
```

### 6.2 待确原因分类

根据表3.1.1的结构，待确产值主要原因可分为以下4类：

| 编号 | 类型 | 内容示例 | 说明 |
|-----|------|---------|------|
| [10] | 实际形象进度 | "已完成主体结构10层" | 形象进度描述 |
| [20] | 合同付款摘要 | "合同约定付款比例70%" | 合同付款条款 |
| [30] | 待确主要原因 | "业主审核滞后" | 待确核心原因 |
| [40] | 待确金额 | 200000.00元 | **待确产值金额** |

### 6.3 附件要求

根据E8的说明，待确金额必须提供以下附件支撑：
1. **量计算文件** - 工程量计算过程
2. **价计算文件** - 价格计算依据
3. **说明**: 须提供与金额相对应的量、价计算文件作为附件支撑，否则对金额不认可

## 九、报表特点总结

### 9.1 优势特点

1. **结构极简** - 仅8行5列，非常简洁
2. **逻辑清晰** - 四项内容逐项说明
3. **定位明确** - 专门用于待确产值原因分析
4. **附件要求严格** - 确保待确金额有充分依据

### 9.2 注意事项

1. **依赖性强** - 必须配合表3.1使用
2. **附件关键** - 附件支撑直接影响待确金额的认可度
3. **数据单一** - 仅提供定性分析和金额数据
4. **需要配合** - 与表3.1产值数据配合使用

### 9.3 使用场景

表3.1.1主要在以下场景使用：
1. **月度汇报** - 月度商务报表的组成部分
2. **产值审核** - 待确认产值的审核依据
3. **风险预警** - 识别产值确权风险
4. **财务对账** - 业主产值与收款对账依据

## 十、与表3.1的关系

### 10.1 表3.1.1在报表体系中的位置

```
表3.1.1(待确产值分析)
    ↓ 被引用
表3.1(业主产值表)
    ↓ 被引用
表3(商务管理汇总表)
    ↓ 被引用

```

### 10.2 数据传递机制

| 传递层级 | 数据内容 | 说明 |
|---------|---------|------|
| 第1层 | C5、C6、C7 | 定性描述 |
| 第2层 | C8 | 定量金额(被引用) |

表3.1.1是整个报表体系中**最小但关键的辅助表**，为待确产值提供定性和定量双重支撑。
