# 180-表3.2字段映射矩阵

## 一、映射关系概述

表3.2的数据主要来源于:
1. **手动填写**(80%的数据)
2. **公式计算**(20%的数据)

表3.2是成本数据的**详细记录表**,数据直接录入后供表3引用。

## 八、SQL查询示例

### 8.1 查询项目成本汇总

```sql
SELECT 
    cost_code as 成本代码,
    SUM(cost_confirmed_acc) as 累计已确认成本,
    SUM(cost_unconfirmed_acc) as 累计待确认成本,
    SUM(cost_confirmed_acc + cost_unconfirmed_acc) as 累计总成本
FROM tb_project_cost_detail
WHERE project_id = ? 
  AND cost_id IN (SELECT id FROM tb_project_cost WHERE year = ? AND month = ?)
GROUP BY cost_code;
```

### 8.2 查询成本明细

```sql
SELECT 
    c.contract_code as 合同编码,
    c.contract_name as 合同名称,
    c.supplier_name as 供应商,
    c.contract_amount as 合同金额,
    d.cost_confirmed_acc as 已确认成本,
    d.cost_unconfirmed_acc as 待确认成本,
    p.paid_amount as 已付款金额,
    (p.payable_confirmed - p.paid_amount) as 应付未付款
FROM tb_project_cost_detail d
JOIN tb_project_contract c ON d.contract_id = c.id
JOIN tb_project_payment p ON d.contract_id = p.contract_id
WHERE d.project_id = ? AND d.cost_id = ?
ORDER BY c.contract_code;
```

### 8.3 查询付款计划

```sql
SELECT 
    c.contract_name as 合同名称,
    pl.payment_method as 付款方式,
    pl.payment_type as 付款类型,
    pl.plan_amount as 支出计划,
    pl.approved_amount as 批准金额,
    (pl.plan_amount - pl.approved_amount) as 差额
FROM tb_project_payment_plan pl
JOIN tb_project_contract c ON pl.contract_id = c.id
WHERE pl.project_id = ? AND pl.cost_id = ?
ORDER BY c.contract_code;
```
