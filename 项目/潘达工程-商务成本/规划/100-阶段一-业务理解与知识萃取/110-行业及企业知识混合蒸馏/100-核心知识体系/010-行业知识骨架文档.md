## 一、业务全景图

### 1.1 行业定位
本项目以**建筑工程行业**（综合型建工企业）为核心定位，业务模型具备行业代表性和通用性。

### 1.2 行业覆盖范围

| 细分领域 | 核心业务 | 本体设计考量 |
|---------|---------|-------------|
| 房屋建筑 | 土建工程、安装工程 | 基础核心，业务最复杂 |
| 市政工程 | 道路、桥梁、管网 | 考虑政府业主特点 |
| 水利水电 | 水利设施、电力设施 | 考虑长周期和特殊成本结构 |
| 公路工程 | 道路、隧道、桥梁 | 与房建有相似性 |
| 铁路工程 | 线路、隧道、桥梁、站场 | 考虑线性工程特点和多专业协同 |

### 1.3 业务全景

```
建筑工程商务成本管理
├── 项目管理
│   ├── 项目立项 → 项目创建 → 项目实施 → 项目竣工
│   └── 项目变更、项目结算、项目决算
├── 合同管理
│   ├── 合同起草 → 合同审批 → 合同签订 → 合同执行
│   └── 合同变更、合同终止
├── 成本管理
│   ├── 预算编制 → 成本归集 → 成本分析 → 成本控制
│   └── 变更签证、成本核算
├── 资金管理
│   ├── 资金计划 → 付款审批 → 收款确认 → 资金监控
│   └── 保函管理、现金流管理
├── 结算审计
│   ├── 月度结算 → 竣工结算 → 审计整改
│   └── 索赔处理
└── 风险管理
    ├── 风险识别 → 风险评估 → 风险处置
    └── 预警机制
```

## 二、核心目标

| 目标类别 | 具体目标 | 量化指标 |
|---------|---------|---------|
| 成本控制 | 实际成本≤预算成本 | 成本偏差率≤3% |
| 利润管理 | 利润率≥目标利润率 | 项目利润率≥8% |
| 资金管理 | 资金周转速度快 | 资金周转率≥4次/年 |
| 风险控制 | 风险损失最小化 | 风险损失率≤2% |

## 四、企业特有规则集成

根据交叉验证结果，以下企业特有规则需要优先实现：

| 规则名称 | 规则描述 | 优先级 | 平台映射 |
