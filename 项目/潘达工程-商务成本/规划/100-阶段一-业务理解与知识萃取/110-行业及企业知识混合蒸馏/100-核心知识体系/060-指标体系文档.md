```

```

## 八、标段级指标（Section-Level）

> **说明**：标段是大型项目划分的独立施工单元，是成本核算和进度管理的重要层级。以下指标用于标段级别的精细化管理。

### 8.1 标段级指标清单

| 指标编号 | 指标名称 | 计算公式 | 数据类型 | 预警阈值 | 决策价值 | 平台映射 |
|---------|---------|---------|---------|---------|---------|---------|
| SI001 | 标段产值完成率 | 标段累计产值 / 标段计划产值 | decimal(5,2) | <80%预警 | 标段进度监控 | Cube度量 |
| SI002 | 标段成本控制率 | 标段实际成本 / 标段目标成本 | decimal(5,2) | >100%预警 | 标段成本管控 | Cube度量 |
| SI003 | 标段进度偏差率 | (标段实际进度 - 标段计划进度) / 标段计划进度 | decimal(5,2) | >±10%预警 | 标段进度偏差 | 派生属性 |
| SI004 | 标段确权产值比 | 标段已确权产值 / 标段总产值 | decimal(5,2) | <85%预警 | 标段确权管理 | Cube度量 |
| SI005 | 标段成本刚性度 | 标段已确权成本 / 标段总成本 | decimal(5,2) | <95%预警 | 标段成本确认 | 派生属性 |
| SI006 | 标段利润贡献率 | 标段利润 / 项目总利润 | decimal(5,2) | - | 标段贡献分析 | Cube度量 |
| SI007 | 标段资金回收率 | 标段已收款 / 标段确权产值 | decimal(5,2) | <75%预警 | 标段回款管理 | 派生属性 |
| SI008 | 标段综合评分 | 产值进度×25% + 成本控制×30% + 利润×25% + 资金×20% | decimal(5,2) | <60预警 | 标段综合评估 | 派生属性 |

### 8.2 标段级指标关系

```
标段产值完成率 = 标段累计产值 / 标段计划产值
标段成本控制率 = 标段实际成本 / 标段目标成本
标段进度偏差率 = (标段实际进度 - 标段计划进度) / 标段计划进度
标段成本刚性度 = 标段已确权成本 / 标段总成本
标段综合评分 = 产值完成率×25% + (1-|成本偏差率|)×30% + 利润率×25% + 收款率×20%
```

### 8.4 标段指标与项目指标的关系

| 关系 | 说明 |
|-----|------|
| 项目产值 = Σ(各标段产值) | 项目产值由各标段产值汇总 |
| 项目成本 = Σ(各标段成本) | 项目成本由各标段成本汇总 |
| 项目进度 = Σ(各标段进度×权重) | 项目进度按标段权重加权计算 |
| 标段进度权重 | section.progress_weight 定义各标段在项目中的权重 |

### 7.2 指标计算逻辑

**项目健康度计算：**
```
项目健康度 = 
  产值完成率 × 20% + 
  (1 - |成本偏差率|) × 30% + 
  利润率 × 25% + 
  收款率 × 15% + 
  (1 - 风险发生率) × 10%
```

**合同执行综合评分：**
```
合同执行综合评分 = 
  履约进度 × 30% + 
  质量达标率 × 25% + 
  (1 - |成本偏差率|) × 25% + 
  安全达标率 × 20%
```
