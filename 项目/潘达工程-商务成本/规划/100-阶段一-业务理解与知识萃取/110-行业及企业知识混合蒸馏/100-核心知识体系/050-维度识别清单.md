## 六、预警规则体系（基于报表编制说明）

### 6.1 项目预警规则

| 预警级别 | 触发条件 | 说明 | 响应措施 |

| 红色预警 | 7项指标中≥6项超范围 **或** 总利润率连续≥3个月为负 | 严重风险 | 立即上报集团，专项整改 |
| 黄色预警 | 7项指标中2-5项超范围 **或** 总利润率连续≥2个月为负 | 中等风险 | 区域公司重点关注，制定改进计划 |
| 绿色正常 | 指标正常 **且** 利润率为正 | 运行良好 | 常规监控 |

| 指标 | 正常范围 | 黄色预警 | 红色预警 |

## 七、平台映射说明

| 维度类型 | 平台能力 | 实施方式 | 说明 |
|---------|---------|---------|------|
| 维度表 | Dimension Type | 通过`define_dimension_type()`定义 | 定义维度元数据结构 |
| 层级结构 | Hierarchy | 通过父子关系字段实现 | 支持组织、成本科目等层级 |
| 维度关联 | Fact-Dimension Join | 通过外键关联事实表 | 建立星型模型关联 |
| 指标计算 | Metric Calculation | 通过dim_indicator定义计算规则 | 支持自定义计算公式 |
| 预警规则 | Alert Rule | 通过dim_warning_level + 阈值配置实现 | 支持多级预警 |
| 实体映射 | Object Type | 映射为平台实体类型 | Project/Contract/Owner等 |
| 关系映射 | Link Type | 定义实体间关系 | 项目-合同、合同-分包商等 |
