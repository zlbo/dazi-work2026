## 二、合同管理诀窍

### CM001 合同条款隐藏风险识别

| 项目 | 内容 |
|------|------|
| **诀窍名称** | 合同条款隐藏风险识别 |
| **适用场景** | 合同谈判、合同评审阶段 |
| **关联流程** | P006（合同审批流程） |
| **规则描述** | 重点关注合同中的"开口条款"：如材料价格调整条款、工期顺延条款、违约责任条款。特别注意：<br>1. 材料价格风险范围是否明确<br>2. 不可抗力定义是否合理<br>3. 争议解决方式是否有利 |
| **业务影响** | 识别潜在风险点，避免后期索赔纠纷，直接影响项目利润 |
| **关联规则** | BR-E003（风险三色预警规则） |
| **实现建议** | 在合同实体中增加`risk_level`属性，通过规则引擎自动识别高风险条款 |
| **平台映射** | Object Type: Contract → Attribute: risk_level；派生属性计算规则 |

### CM002 合同价格策略

| 项目 | 内容 |
|------|------|
| **诀窍名称** | 合同价格策略 |
| **适用场景** | 投标报价、合同谈判阶段 |
| **关联流程** | P002（项目投标流程）、P006（合同审批流程） |
| **规则描述** | 根据项目特点选择合适的合同计价方式：<br>1. 总价合同：适用于设计明确、风险可控的项目<br>2. 单价合同：适用于设计变更可能性大的项目<br>3. 成本加酬金合同：适用于风险较高的项目 |
| **业务影响** | 合理选择合同类型可有效转移风险，保障利润空间 |
| **关联规则** | QR-P001（项目利润率计算规则） |
| **实现建议** | 建立合同类型与项目特征的匹配规则，辅助报价决策 |
| **平台映射** | Object Type: Contract → Attribute: contract_type；派生属性：risk_transfer_score |

### CM003 履约保函管理

| 项目 | 内容 |
|------|------|
| **诀窍名称** | 履约保函管理 |
| **适用场景** | 合同签订、项目实施阶段 |
| **关联流程** | P006（合同审批流程）、P019（保函管理流程） |
| **规则描述** | 保函管理的关键控制点：<br>1. 保函金额：原则上不超过合同金额的10%<br>2. 保函期限：应覆盖缺陷责任期<br>3. 保函类型：优先选择见索即付保函<br>4. 反担保：要求分包商提供等额反担保 |
| **业务影响** | 合理的保函管理可降低企业资金占用成本，控制担保风险 |
| **关联规则** | FR-C013（保函编码规则） |
| **实现建议** | 在担保实体中增加保函管理属性，设置预警规则 |
| **平台映射** | Object Type: Guarantee → Attributes: guarantee_amount, guarantee_period, guarantee_type |

### CM004 争议条款前置处理

## 三、变更签证诀窍

| 项目 | 内容 |
|------|------|

| **适用场景** | 变更发生后及时办理签证 |
| **关联流程** | P009（变更签证流程）、P004（项目变更流程） |
| **规则描述** | 变更签证必须在变更发生后72小时内提出，超过时限业主有权拒绝。关键动作：<br>1. 第一时间拍照取证<br>2. 填写变更签证单<br>3. 同步抄送相关部门<br>4. 跟踪签字进度 |
| **业务影响** | 及时办理签证可确保变更费用得到确认，避免成本损失 |
| **关联规则** | BR-C015（变更签证时效规则） |
| **实现建议** | 设置变更签证时限预警，超过48小时未办理自动提醒 |
| **平台映射** | Action Type: A021（ApplyChange）；派生属性: change_application_delay_days；预警规则 |

| **业务影响** | 合理的计价策略可最大化变更收益 |
| **关联规则** | QR-C003（变更成本计算规则） |
