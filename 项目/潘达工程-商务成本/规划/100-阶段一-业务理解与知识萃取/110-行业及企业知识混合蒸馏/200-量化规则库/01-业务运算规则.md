## 一、成本类规则（QR-C）

### 【QR-C001】成本偏差率计算规则
- **规则名称**：成本偏差率计算规则
- **规则类别**：业务运算规则 → 成本类规则
- **适用实体**：ProjectCost, Budget
- **适用场景**：成本分析、成本预警
- **量化公式**：成本偏差率 = (实际成本 - 预算成本) / 预算成本 × 100%
- **参数定义**：
  - 实际成本：来自成本台账 fact_project_cost
  - 预算成本：来自预算表 fact_budget
- **阈值定义**：成本偏差率 > 5% 时触发预警
- **预警级别**：黄色预警（5%-10%）、红色预警（>10%）
- **业务依据**：行业成本控制标准，5%为合理偏差范围
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_cost, fact_budget
- **平台映射**：派生属性（成本偏差率）+ Action Type（预警通知）
- **关联指标**：成本偏差率指标
- **关联关系**：成本与预算的对比关系

- **规则名称**：成本节约率计算规则
- **规则类别**：业务运算规则 → 成本类规则

- **参数定义**：

  - 预算成本：来自预算表 fact_budget
- **阈值定义**：成本节约率 > 5% 时视为优秀
- **预警逻辑**：无（正向指标）
- **业务依据**：成本控制优秀标准
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_cost, fact_budget
- **平台映射**：派生属性（成本节约率）
- **关联指标**：成本节约率指标

### 【QR-C003】材料成本占比计算规则
- **规则名称**：材料成本占比计算规则
- **规则类别**：业务运算规则 → 成本类规则
- **适用实体**：ProjectCost
- **适用场景**：成本结构分析
- **量化公式**：材料成本占比 = 材料成本 / 直接成本 × 100%
- **参数定义**：
  - 材料成本：来自成本台账的材料费科目（5403开头）
  - 直接成本：人工+材料+机械+其他直接费
- **阈值定义**：材料成本占比 > 65% 或 < 50% 时触发预警
- **预警逻辑**：材料占比过高可能存在材料浪费，过低可能存在漏项
- **业务依据**：建筑行业直接成本结构：材料50%-65%、人工20%-30%、机械8%-12%、其他2%-5%
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_cost
- **平台映射**：派生属性（材料成本占比）+ Action Type（预警通知）
- **关联指标**：材料成本占比指标

### 【QR-C006】间接成本占比计算规则

- **规则类别**：业务运算规则 → 成本类规则
- **适用实体**：ProjectCost
- **适用场景**：成本结构分析

- **预警逻辑**：间接成本占比过高可能存在管理效率问题，过低可能存在漏项
- **业务依据**：建筑行业间接成本占总成本比例通常为6%-12%
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_cost

### 【QR-C007】成本结构校验规则
- **规则名称**：成本结构校验规则
- **规则类别**：业务运算规则 → 成本类规则
- **适用实体**：ProjectCost
- **适用场景**：成本结构合规性检查
- **量化公式**：Σ(材料占比+人工占比+机械占比+其他直接费占比) = 100%
- **参数定义**：各成本要素占直接成本比例
- **阈值定义**：各要素占比之和 ≠ 100% ± 0.5% 时触发错误
- **预警逻辑**：成本结构不平衡，需检查数据完整性
- **业务依据**：成本核算基本原则，各要素占比之和必须等于100%
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_cost
- **平台映射**：数据校验规则 + Action Type（数据修正通知）
- **关联指标**：成本结构完整性指标

## 二、资金类规则（QR-F）

### 【QR-F001】资金周转率计算规则
- **规则名称**：资金周转率计算规则
- **规则类别**：业务运算规则 → 资金类规则
- **适用实体**：CashFlow, Finance
- **适用场景**：资金效率分析
- **量化公式**：资金周转率 = 主营业务收入 / 平均营运资金
- **参数定义**：
  - 主营业务收入：来自产值确认表
  - 平均营运资金：(期初营运资金 + 期末营运资金) / 2
  - 营运资金 = 流动资产 - 流动负债
- **阈值定义**：资金周转率 < 2次/年时触发预警
- **预警级别**：黄色预警（1.5-2次）、红色预警（<1.5次）
- **业务依据**：建筑行业资金周转普遍较慢，合理区间2-4次/年
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_output, dim_finance
- **平台映射**：派生属性（资金周转率）+ Rule（预警规则）
- **关联指标**：资金周转率指标

### 【QR-F002】收款率计算规则
- **规则名称**：收款率计算规则

- **阈值定义**：收款率 < 80% 时触发预警
- **预警级别**：黄色预警（60%-80%）、红色预警（<60%）
- **业务依据**：正常收款率应≥80%
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_payment, fact_project_output
- **平台映射**：派生属性（收款率）+ Action Type（预警通知）
- **关联指标**：工程款回收率指标

### 【QR-F004】融资成本率计算规则

- **数据来源**：fact_financing, dim_finance

## 三、进度类规则（QR-S）

### 【QR-S001】进度偏差率计算规则
- **规则名称**：进度偏差率计算规则
- **规则类别**：业务运算规则 → 进度类规则
- **适用实体**：ProjectProgress, ProjectPlan
- **适用场景**：进度管理
- **量化公式**：进度偏差率 = (实际进度 - 计划进度) / 计划进度 × 100%
- **参数定义**：
  - 实际进度：来自进度报表
  - 计划进度：来自进度计划
- **阈值定义**：进度偏差率 < -10%（滞后10%）时触发预警
- **预警级别**：黄色预警（-10%~-5%）、红色预警（<-10%）
- **业务依据**：合理进度偏差应控制在±5%以内
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_progress, dim_project_plan
- **平台映射**：派生属性（进度偏差率）+ Action Type（预警通知）
- **关联指标**：进度偏差率指标

### 【QR-S002】工期完成率计算规则
- **规则名称**：工期完成率计算规则
- **规则类别**：业务运算规则 → 进度类规则
- **适用实体**：Project, ProjectPlan
- **适用场景**：工期管理
- **量化公式**：工期完成率 = 已完成天数 / 总工期天数 × 100%
- **参数定义**：
  - 已完成天数：根据开工日期和当前日期计算
  - 总工期天数：来自项目计划
- **阈值定义**：工期完成率 < 进度完成率时触发预警
- **预警逻辑**：工期进度落后于计划时发送预警
- **业务依据**：工期与进度应同步
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：dim_project, dim_project_plan
- **平台映射**：派生属性（工期完成率）
- **关联指标**：工期完成率指标

### 【QR-S003】关键路径进度率计算规则
- **规则名称**：关键路径进度率计算规则
- **规则类别**：业务运算规则 → 进度类规则
- **适用实体**：ProjectProgress
- **适用场景**：关键路径管理
- **量化公式**：关键路径进度率 = 关键路径已完成工作量 / 关键路径总工作量 × 100%
- **参数定义**：
  - 关键路径已完成工作量：来自进度报表
  - 关键路径总工作量：来自项目计划
- **阈值定义**：关键路径进度率 < 计划进度率时触发预警
- **预警级别**：红色预警（关键路径延误）
- **业务依据**：关键路径延误将直接影响总工期
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_progress, dim_project_plan

## 四、风险类规则（QR-R）

### 【QR-R001】风险综合评分规则
- **规则名称**：风险综合评分规则
- **规则类别**：业务运算规则 → 风险类规则
- **适用实体**：Risk
- **适用场景**：风险评估
- **量化公式**：风险综合评分 = (发生概率 × 10) × 影响程度 / 可检测性修正系数
- **参数定义**：

  - 影响程度：1-5（1=轻微，5=严重）

- **阈值定义**：风险综合评分 > 25 时触发预警
- **预警级别**：黄色预警（25-40）、红色预警（>40）
- **业务依据**：风险矩阵评估方法，最高评分=10×5/0.8=62.5分
- **企业规模适配**：年产值500-600亿企业适用

- **规则类别**：业务运算规则 → 利润类规则
- **适用实体**：ProjectOutput, ProjectCost, Finance
- **适用场景**：价值创造分析
- **量化公式**：EVA = 税后净营业利润 - 资本成本

- **阈值定义**：EVA < 0 时触发预警
- **预警逻辑**：EVA为负表明未创造价值
- **业务依据**：经济增加值衡量价值创造能力
- **企业规模适配**：年产值500-600亿企业适用
- **数据来源**：fact_project_output, fact_project_cost, dim_finance
- **平台映射**：派生属性（EVA）+ Action Type（预警通知）
