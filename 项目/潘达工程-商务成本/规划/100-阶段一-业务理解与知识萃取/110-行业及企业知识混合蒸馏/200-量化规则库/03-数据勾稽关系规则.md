## 一、指标间勾稽关系规则（CR-I）

### 【CR-I001】利润计算公式规则
- **规则编号**：CR-I001
- **规则名称**：利润计算公式规则
- **规则类别**：数据勾稽规则 → 指标间勾稽
- **适用实体**：Project
- **勾稽关系**：利润 = 产值 - 成本
- **公式表达**：Profit = Output - Cost
- **参数定义**：
  - Profit：项目利润额
  - Output：已确认产值
  - Cost：已确认成本
- **验证逻辑**：精确相等
- **业务依据**：基本利润计算公式
- **数据来源**：fact_project_output, fact_project_cost
- **校验规则**：Profit == Output - Cost
- **错误级别**：ERROR
- **错误提示**：利润计算与产值、成本不匹配
- **企业规模适配**：年产值500-600亿企业适用
- **平台映射**：数据校验规则

- **规则编号**：CR-I013
- **规则名称**：含税产值与不含税产值勾稽规则
- **规则类别**：数据勾稽规则 → 指标间勾稽
- **适用实体**：ProjectOutput
- **勾稽关系**：含税产值 = 不含税产值 × (1 + 税率)

- **验证逻辑**：精确相等（允许±0.01%误差）
- **业务依据**：增值税计算规则
- **数据来源**：fact_project_output
- **校验规则**：TaxIncludedOutput == TaxExcludedOutput × (1 + TaxRate)
- **错误级别**：ERROR
- **错误提示**：含税产值与不含税产值计算不匹配
- **企业规模适配**：年产值500-600亿企业适用
- **平台映射**：数据校验规则

### 【CR-I014】增值税计算勾稽规则
- **规则编号**：CR-I014
- **规则名称**：增值税计算勾稽规则
- **规则类别**：数据勾稽规则 → 指标间勾稽
- **适用实体**：ProjectOutput, Finance
- **勾稽关系**：应交增值税 = 销项税额 - 进项税额
- **公式表达**：VAT = OutputTax - InputTax
- **参数定义**：
  - VAT：应交增值税
  - OutputTax：销项税额 = 不含税产值 × 税率
  - InputTax：进项税额 = 不含税成本 × 税率
- **验证逻辑**：精确相等
- **业务依据**：增值税抵扣规则
- **数据来源**：fact_project_output, fact_project_cost, dim_finance
- **校验规则**：VAT == OutputTax - InputTax
- **错误级别**：ERROR
- **错误提示**：应交增值税计算与销项、进项税额不匹配
- **企业规模适配**：年产值500-600亿企业适用
- **平台映射**：数据校验规则

- **规则编号**：CR-T008

- **规则类别**：数据勾稽规则 → 时间序列勾稽
- **适用实体**：ProjectOutput, ProjectProgress
- **勾稽关系**：本期产值 ≈ 合同金额 × 本期进度比例
- **公式表达**：CurrentOutput ≈ ContractAmount × ProgressRatio
- **参数定义**：
  - CurrentOutput：本期产值
  - ContractAmount：合同金额
  - ProgressRatio：本期进度比例（0-1）
- **验证逻辑**：近似相等（允许±5%误差）
- **业务依据**：产值与进度的匹配关系
- **数据来源**：fact_project_output, fact_project_progress
- **校验规则**：|CurrentOutput - ContractAmount × ProgressRatio| / ContractAmount × ProgressRatio <= 0.05
- **错误级别**：WARNING
- **错误提示**：本期产值与进度比例不匹配
- **企业规模适配**：年产值500-600亿企业适用
