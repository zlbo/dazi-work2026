# 建筑工程行业商务成本管理核心实体清单

## 一、管理对象类实体

【E001】项目
- 业务定义：具有独立立项编号的建筑工程项目
- 核心属性：项目名称、项目编号、项目类型、开工日期、竣工日期、项目状态、合同金额、项目经理、项目结构（主体/二次结构）、工程阶段（基础/主体/装修/竣工）
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：包含合同、成本、产值、风险、标段、工程量清单等实体

【E002】标段
- 业务定义：大型项目划分的独立施工单元
- 核心属性：标段编号、标段名称、标段范围、中标单位、中标金额、开工日期、竣工日期、招标方式、评标结果
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，包含合同、成本、工程量清单

【E003】合同
- 业务定义：与业主或供应商签订的具有法律效力的协议
- 核心属性：合同编号、合同名称、合同金额、签订日期、有效期、合同类型（总包/分包/采购/劳务）、甲方、乙方、垫资比例、付款节点、质保金比例、履约保函要求
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目或标段，包含合同项

【E004】分包合同
- 业务定义：总承包人与分包商签订的工程分包协议
- 核心属性：分包合同编号、总包合同编号、分包范围、分包金额、专业类别、资质要求、安全责任、进度要求
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属总包合同，关联分包商

【E005】劳务合同
- 业务定义：与劳务分包公司签订的劳务用工协议
- 核心属性：劳务合同编号、劳务班组、工种类型、人数、单价、工期、社保要求、安全责任
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联分包商

【E006】合同项
- 业务定义：合同中的具体工作内容或产品条目
- 核心属性：合同项编号、合同项名称、单价、数量、金额、计量单位、工作内容、材料类型（主材/辅材/周转材料）、工程量清单编号
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属合同，关联成本

【E007】预算
- 业务定义：项目的成本计划和控制目标
- 核心属性：预算编号、预算名称、预算金额、编制日期、生效日期、预算版本、编制部门、核算层级（二级/三级）、预算类型（概算/预算/标底）
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，对比实际成本

【E008】变更签证
- 业务定义：项目实施过程中的设计变更或现场签证
- 核心属性：变更编号、变更原因、变更内容、变更金额、审批状态、变更日期、责任方、索赔金额
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，影响成本和产值

【E009】工程量清单
- 业务定义：项目的工程数量和单价明细
- 核心属性：清单编号、项目名称、项目特征、计量单位、工程量、综合单价、合价、清单规范版本
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目或标段，关联合同项

【E010】融资
- 业务定义：项目的资金筹措和贷款安排
- 核心属性：融资编号、融资方式（银行贷款/债券/保理）、融资金额、利率、期限、还款计划、担保方式、利息支出
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，产生利息支出

【E011】材料库存
- 业务定义：项目使用的材料库存管理
- 核心属性：库存编号、材料名称、规格型号、库存数量、单价、金额、存放地点、材料类别（主材/辅材/周转材料）、供应商
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联成本

【E012】机械设备
- 业务定义：项目使用的施工机械和设备
- 核心属性：设备编号、设备名称、型号规格、原值、折旧方法、台班单价、使用状态、维护周期、设备状态（自有/租赁）
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联机械费、关联成本

【E013】投标保证金
- 业务定义：投标人向招标人提供的保证金，确保中标后签订合同
- 核心属性：保证金编号、招标项目、投标单位、保证金金额、缴纳日期、退回日期、退回状态、没收状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联招标流程，影响现金流

【E014】履约保证金
- 业务定义：承包商向发包人提供的保证金，确保承包商履行合同义务
- 核心属性：保证金编号、合同编号、保证金金额、缴纳日期、到期日期、返还条件、已返还金额、未返还金额、没收状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属合同，影响资金占用，关联现金流

【E015】质量保证金（质保金）
- 业务定义：合同中约定的质量保证金，用于保证工程质量问题
- 核心属性：质保金编号、合同编号、质保金金额、质保期（起止日期）、返还条件、已返还金额、未返还金额、质保期满状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属合同，影响现金流，关联结算

【E016】农民工工资保证金
- 业务定义：专门用于支付农民工工资的资金保障
- 核心属性：保证金编号、项目、保证金金额、缴纳日期、开户银行、动用状态、动用金额、补缴状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，影响现金流，关联劳务成本

【E017】履约保函
- 业务定义：保证合同履行的担保文件（银行保函形式）
- 核心属性：保函编号、合同编号、保函金额、保函类型（投标/履约/预付款）、有效期、出具银行、受益人、保函状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属合同，影响资金占用，关联保证金

【E018】押金
- 业务定义：各种场合收取的押金（如设备租赁押金、场地押金等）
- 核心属性：押金编号、项目、押金类型（设备租赁/场地租赁/其他）、押金金额、收取日期、退还条件、已退还金额、未退还金额
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联成本，影响现金流

【E019】罚款
- 业务定义：因违约、违规等产生的罚款记录
- 核心属性：罚款编号、项目、罚款类型（违约罚款/违规罚款/质量安全罚款/其他）、罚款金额、罚款原因、开具单位、罚款日期、缴纳状态、申诉状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联成本，关联合同，影响现金流

【E020】赔偿
- 业务定义：因合同违约或侵权产生的赔偿记录
- 核心属性：赔偿编号、项目、赔偿类型（合同赔偿/侵权赔偿/工伤赔偿/其他）、赔偿金额、赔偿原因、责任方、赔偿日期、支付状态
- 实体类型：管理对象
- 适用领域：通用
- 业务关联：归属项目，关联合同，关联索赔，影响现金流

## 二、业务活动类实体

【E021】产值
- 业务定义：已完成的工程量对应的金额
- 核心属性：产值编号、项目、确认金额、待确金额、确认日期、业主确认状态、工程量完成比例、进度偏差
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联合同

【E022】成本
- 业务定义：项目实施过程中发生的各项费用
- 核心属性：成本编号、项目、成本类型（人工费/材料费/机械费/管理费/措施费/其他）、成本金额、发生日期、成本状态、成本层级（主控/分项/明细）、责任部门
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联合同项

【E023】付款
- 业务定义：向供应商或分包商支付款项的记录
- 核心属性：付款编号、项目、合同、付款金额、付款日期、付款方式、付款状态、付款类型（预付款/进度款/结算款/质保金）
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联合同

【E024】收款
- 业务定义：从业主收到的款项记录
- 核心属性：收款编号、项目、合同、收款金额、收款日期、收款方式、收款类型（预付款/进度款/结算款/质保金）、逾期天数
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联合同

【E025】发票
- 业务定义：供应商开具的费用凭证
- 核心属性：发票编号、发票金额、开票日期、发票类型（增值税专票/普票）、税额、发票状态、认证状态
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属合同，关联付款

【E026】结算
- 业务定义：项目或合同的最终价款结算
- 核心属性：结算编号、项目、结算金额、结算日期、审核状态、审计意见、三算对比（概算/预算/决算）、审减金额
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联合同

【E027】现金流
- 业务定义：项目资金的流入流出记录
- 核心属性：现金流编号、项目、流入金额、流出金额、余额、发生日期、资金类型（经营/投资/筹资）、资金来源
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联付款和收款

【E028】库存盘点
- 业务定义：对项目材料库存的定期盘点
- 核心属性：盘点编号、项目、盘点日期、盘点类型（定期/专项）、盘盈金额、盘亏金额、差异原因、盘点责任人
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联材料库存

【E029】材料调拨
- 业务定义：项目间或部门间的材料转移
- 核心属性：调拨编号、调出项目、调入项目、材料名称、数量、单价、调拨日期、调拨原因、审批状态
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联材料库存

【E030】机械费计算
- 业务定义：施工机械使用费用的核算
- 核心属性：计算编号、项目、机械类型、工作时长、台班数、单价、金额、油料消耗、维修费用
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联成本

【E031】废旧物资处理
- 业务定义：项目废旧物资的处置和回收
- 核心属性：处理编号、项目、物资名称、数量、原值、残值、处置方式（变卖/报废/回收）、回收金额、处理日期
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联成本

【E032】工程量计量
- 业务定义：对已完成工程量的测量和确认
- 核心属性：计量编号、项目、合同项、计量数量、计量日期、监理确认状态、业主确认状态
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联合同项

【E033】索赔
- 业务定义：因合同变更或风险事件提出的费用补偿请求
- 核心属性：索赔编号、项目、索赔原因、索赔金额、索赔依据、审批状态、索赔结果、责任认定
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联变更签证

【E034】保证金收取
- 业务定义：向分包商或供应商收取的保证金
- 核心属性：保证金编号、项目、收款单位、保证金类型（履约保证金/质量保证金/其他）、保证金金额、收取日期、退还条件、已退还金额、未退还金额
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联分包合同，影响现金流

【E035】罚款收取
- 业务定义：向分包商或供应商收取的违约罚款
- 核心属性：罚款编号、项目、罚款对象、罚款类型（违约罚款/质量罚款/安全罚款）、罚款金额、罚款原因、开具日期、缴纳状态
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，关联分包合同，关联成本

【E036】工程保险
- 业务定义：项目投保的各类工程保险
- 核心属性：保险编号、项目、保险类型（工程险/意外险/责任险）、保险公司、保险金额、保费金额、投保日期、到期日期、理赔状态
- 实体类型：业务活动
- 适用领域：通用
- 业务关联：归属项目，影响成本，关联风险

## 三、参与方类实体

【E037】发包人（业主）
- 业务定义：项目的投资方或建设单位，是工程合同的甲方
- 核心属性：发包人名称、发包人编号、统一社会信用代码、联系人、联系方式、所属行业、资金来源
- 实体类型：参与方
- 适用领域：通用
- 业务关联：签订合同，支付款项，审批变更

【E038】总承包人（施工单位）
- 业务定义：承接项目总承包任务的企业，是工程合同的乙方
- 核心属性：总承包人名称、总承包人编号、资质等级、项目经理、安全许可证、信用等级
- 实体类型：参与方
- 适用领域：通用
- 业务关联：签订合同，实施工程，管理分包

【E039】供应商
- 业务定义：提供材料、设备或服务的企业
- 核心属性：供应商名称、供应商编号、统一社会信用代码、联系人、联系方式、资质等级、信用等级、供货范围
- 实体类型：参与方
- 适用领域：通用
- 业务关联：签订合同，提供发票，供应物资

【E040】分包商
- 业务定义：承接部分工程施工的企业（含劳务分包，专业分包）
- 核心属性：分包商名称、分包商编号、资质等级、专业类别、项目经理、分包类型（劳务/专业）、安全许可证
- 实体类型：参与方
- 适用领域：通用
- 业务关联：签订分包合同，完成工程，提供发票

【E041】监理单位
- 业务定义：受发包人委托对项目进行监督管理的机构
- 核心属性：监理单位名称、监理单位编号、资质等级、项目总监、联系方式、监理范围
- 实体类型：参与方
- 适用领域：通用
- 业务关联：监督项目，出具监理报告，确认工程量

【E042】设计院（设计单位）
- 业务定义：承担项目设计任务的机构
- 核心属性：设计院名称、设计院编号、资质等级、专业范围、项目负责人、设计资质类型
- 实体类型：参与方
- 适用领域：通用
- 业务关联：提供设计图纸，参与设计变更，出具设计交底

【E043】监管部门
- 业务定义：对项目进行监督管理的政府或行业机构
- 核心属性：部门名称、部门编号、监管范围、联系方式、监管职责、监管级别
- 实体类型：参与方
- 适用领域：通用
- 业务关联：监督项目合规性，出具监管意见，验收项目

【E044】外部协助单位
- 业务定义：为项目提供专业技术支持的外部机构（如检测机构、咨询公司、造价事务所等）
- 核心属性：单位名称、单位编号、专业领域、资质等级、服务内容、收费标准
- 实体类型：参与方
- 适用领域：通用
- 业务关联：提供专业服务，出具技术报告，审计结算

【E045】造价咨询单位
- 业务定义：提供工程造价咨询服务的机构
- 核心属性：单位名称、单位编号、资质等级、项目负责人、咨询范围、收费标准
- 实体类型：参与方
- 适用领域：通用
- 业务关联：编制预算，审核结算，提供造价咨询

【E046】招标代理机构

- 实体类型：参与方
- 适用领域：通用
- 业务关联：组织招标，编制招标文件，评标服务

【E047】劳务公司

- 实体类型：参与方
- 适用领域：通用
- 业务关联：签订劳务合同，提供劳务人员，管理农民工工资

【E048】保险公司

- 实体类型：参与方

【E049】部门

【E050】员工

- 实体类型：参与方
- 适用领域：通用
- 业务关联：负责项目执行，审批业务，记录操作

## 四、管理要素类实体

【E051】指标
- 业务定义：用于衡量项目绩效的量化指标（含预实比、实目比、三算指标）
- 核心属性：指标编号、指标名称、指标值、计算规则、预警阈值、数据来源、指标类型（预实比/实目比/比率/金额/偏差率）、统计周期
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：评估项目，触发预警，支持决策分析

【E052】风险
- 业务定义：可能影响项目目标实现的不确定因素
- 核心属性：风险编号、风险名称、风险描述、风险等级（高/中/低）、发生概率、影响程度、应对措施、风险状态
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：归属项目，触发预警，支持风险管控

【E053】文档
- 业务定义：项目相关的各类文件资料
- 核心属性：文档编号、文档名称、文档类型、版本、上传日期、存储位置、文档状态、密级
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：归属项目，支持业务审批，提供审计依据

【E054】审批流程
- 业务定义：业务事项的审批流程定义
- 核心属性：流程编号、流程名称、审批节点、审批权限、流转规则、驳回规则、超时处理
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：审批各类业务事项，控制业务流转

【E055】数据字典
- 业务定义：业务数据的标准化编码和定义
- 核心属性：字典编号、字典名称、字典值、编码规则、描述说明、生效日期、失效日期
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：规范数据录入，保证数据一致性，支持数据治理

【E056】地区维度
- 业务定义：项目所在的地理区域划分
- 核心属性：地区编号、地区名称、地区层级（境内/境外、大区/省份/城市/区县）、时区、税务政策、市场行情、人工成本水平
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：归属项目，影响成本和税务，支持区域分析

【E057】日期维度
- 业务定义：时间维度的标准化管理
- 核心属性：日期、年、季度、月、周、工作日/休息日、财务周期、工程进度节点、节假日
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：关联所有业务活动，支持时间维度分析，支持进度跟踪

【E058】核算模式
- 业务定义：企业成本核算的组织模式
- 核心属性：核算模式编号、核算模式名称（二级核算/三级核算）、核算层级（公司/分公司/项目）、责任主体、核算规则、成本归集方式
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：定义成本核算方式，支持成本管控

【E059】成本要素
- 业务定义：构成项目成本的基本要素分类
- 核心属性：要素编号、要素名称、要素类别（人工费/材料费/机械费/管理费/措施费/其他）、计算规则、成本科目、预算科目
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：归属成本，支持成本分析，定义成本结构

【E060】预警规则
- 业务定义：触发预警的条件和规则
- 核心属性：规则编号、规则名称、预警条件、阈值设置、预警级别、通知对象、处理流程
- 实体类型：管理要素
- 适用领域：通用
- 业务关联：监控业务指标，触发预警，支持风险预警

【E061】项目模板
- 业务定义：项目管理的标准化模板

- 实体类型：管理要素
- 适用领域：通用
- 业务关联：新建项目时引用，保证项目管理标准化
