## 二、L1：基础业务知识（行业通用）

### 2.1 核心业务实体清单

**核心业务实体是"业务世界"中的基本对象。以下为房建行业商务成本管理领域的通用业务实体清单。

| 序号 | 业务实体 | 定义 | 典型属性 | 典型状态 |
|------|---------|------|---------|---------|
| 1 | 项目 | 具有独立立项编号的建筑工程项目 | 项目名称、项目编号、项目类型、开工日期、竣工日期、合同金额、建筑面积、项目地点、业主单位、施工单位、监理单位 | 立项中、在建、竣工、结算中、已结算、已决算 |
| 2 | 标段 | 项目的细分部分，通常按专业或区域划分 | 标段名称、标段编号、标段类型、合同金额、开工日期、竣工日期 | 在建、竣工、结算中、已结算 |
| 3 | 产值 | 已完成工程量对应的金额 | 确认产值、待确产值、累计产值、当期产值、产值类型、产值日期 | 已确认、待确认、已审核、未审核 |
| 4 | 成本 | 项目实施过程中发生的各项费用 | 人工费、材料费、机械费、管理费、措施费、规费、税金、其他费用、累计成本、当期成本、成本类型、成本日期 | 已确认、待确认、已支付、未支付 |
| 5 | 利润 | 项目收入减去成本后的余额 | 利润总额、净利润、利润率、累计利润、当期利润 | 盈利、亏损、持平 |
| 6 | 资金流 | 项目资金的流入和流出 | 流入：进度款、预付款、质保金返还、变更签证款；流出：成本支付、税费、管理费、保证金、违约金 | 流入、流出、待审批、已审批 |
| 7 | 合同 | 项目执行的法律依据 | 合同类型、合同编号、合同名称、合同金额、签订日期、生效日期、终止日期、甲方单位、乙方单位、合同状态 | 执行中、已完成、已终止、已解除 |
| 8 | 风险 | 项目执行中的潜在问题 | 风险类型、风险等级、风险金额、发生概率、影响程度、风险状态 | 已发生、潜在、已解决、待解决 |
| 9 | 供应商 | 提供产品或服务的外部单位 | 供应商名称、供应商编号、供应商类型、资质等级、合作状态、联系人、联系方式 | 合作中、已停用、待审核、已审核 |
| 10 | 资源 | 项目实施所需的各类资源 | 资源类型、资源名称、资源编号、规格型号、单位、单价 | 可用、已使用、待采购、已采购 |
| 11 | 变更签证 | 项目实施过程中的变更和签证 | 变更类型、签证编号、变更内容、变更金额、审批状态、生效日期 | 已审批、待审批、已生效、未生效 |
| 12 | 结算 | 项目或标段的最终费用结算 | 结算类型、结算编号、结算金额、结算日期、审批状态、生效日期 | 已完成、待审批、已审批、已生效 |
| 13 | 预算 | 项目实施前的费用预算 | 预算类型、预算编号、预算金额、编制日期、审批状态、生效日期 | 已批准、待审批、已生效、已调整 |
| 14 | 指标 | 项目管理的关键绩效指标 | 指标名称、指标编号、指标类型、指标值、计算规则、预警阈值 | 正常、预警、异常、待更新 |

### 2.2 核心业务流程清单

以下为房建行业商务成本管理领域的通用核心业务流程。

| 序号 | 业务流程 | 包含阶段 | 关键指标 | 行业通用性 |
|------|---------|---------|---------|---------|
| 1 | 项目生命周期管理 | 项目立项 → 初步设计 → 施工图设计 → 施工准备 → 施工阶段 → 竣工验收 → 结算 → 决算 → 保修 | 项目周期、预算偏差率、成本节约率、竣工验收合格率 | 通用（房建行业） |
| 2 | 成本全流程管理 | 成本预算编制 → 成本目标分解 → 成本过程控制 → 成本核算 → 成本分析 → 成本考核 → 成本优化 | 预算偏差率、成本节约率、成本利润率、单位面积成本 | 通用（建筑行业） |
| 3 | 资金全流程管理 | 资金预算编制 → 资金筹措 → 资金支付计划 → 资金支付审批 → 资金支付 → 资金回笼 → 资金分析 → 资金优化 | 资金周转率、现金流量、资金占用成本、资金到位率 | 通用（企业通用） |
| 4 | 合同全生命周期管理 | 合同策划 → 合同谈判 → 合同签订 → 合同执行 → 合同变更 → 合同结算 → 合同归档 | 合同履约率、合同变更率、合同纠纷率、合同毛利率 | 通用（企业通用） |
| 5 | 产值全流程管理 | 产值申报 → 产值审核 → 产值确认 → 产值统计 → 产值分析 | 产值完成率、产值增长率、产值偏差率、确认产值比例 | 通用（房建行业） |
| 6 | 供应商全生命周期管理 | 供应商准入 → 供应商评估 → 供应商选择 → 供应商合作 → 供应商考核 → 供应商退出 | 供应商合格率、供应商满意度、供应商交付及时率、供应商成本率 | 通用（企业通用） |
| 7 | 风险全流程管理 | 风险识别 → 风险评估 → 风险应对 → 风险监控 → 风险处置 → 风险复盘 | 风险发生率、风险损失率、风险应对及时率、风险解决率 | 通用（企业通用） |
| 8 | 变更签证全流程管理 | 变更申请 → 变更评估 → 变更审批 → 变更执行 → 变更确认 → 变更结算 | 变更发生率、变更金额占比、变更审批周期、变更结算率 | 通用（房建行业） |

### 2.3 核心术语表

| 术语 | 定义 | 业务含义 | 行业通用性 |
|------|------|---------|---------|
| 确认产值 | 已经业主签字确认的产值 | 可计入收入，影响利润计算 | 通用（房建行业） |
| 待确产值 | 尚未得到业主确认的产值 | 暂不能计入收入 | 通用（房建行业） |
| 刚性成本 | 已经确认的、不可变更的成本 | 不可变更，影响利润计算 | 通用（建筑行业） |
| 弹性成本 | 尚未确认的、可能变动的成本 | 可能变动，影响利润预测 | 通用（建筑行业） |
| 成本偏差率 | 实际成本与预算的偏差比例 | 衡量成本控制效果 | 通用（企业通用） |
| 资金预算 | 项目资金使用计划 | 指导资金调度 | 通用（企业通用） |
| 变更签证 | 项目实施过程中的设计变更和现场签证 | 房建行业特有术语，影响成本和产值 | 通用（房建行业） |
| 进度款 | 按施工进度支付的工程款 | 房建行业通用术语，影响资金流 | 通用（房建行业） |
| 质保金 | 合同金额的一部分预留金，用于质量保修 | 保修期满后返还，影响资金流 | 通用（建筑行业） |
| 结算款 | 项目或标段竣工后结算的款项 | 最终确认的工程款项，影响利润 | 通用（建筑行业） |
| 决算款 | 项目最终决算的款项 | 包含所有费用的最终款项，影响利润 | 通用（建筑行业） |
| 单位面积成本 | 项目总成本除以建筑面积 | 衡量成本控制效果的关键指标 | 通用（房建行业） |
| 成本利润率 | 利润与成本的比例 | 衡量项目盈利能力的关键指标 | 通用（企业通用） |
| 资金周转率 | 营业收入与平均资金占用的比例 | 衡量资金使用效率的关键指标 | 通用（企业通用） |
| 合同履约率 | 按时履约的合同数量与总合同数量的比例 | 衡量合同管理效果的关键指标 | 通用（企业通用） |
| 产值完成率 | 实际完成产值与计划产值的比例 | 衡量项目进度的关键指标 | 通用（房建行业） |

## 四、L3：隐性业务关系（重点）

### 4.1 隐性业务关系清单

以下为行业通用的隐性业务关系。这些关系不在数据表中可能不直接体现，但对业务决策有重要影响。

**说明：以下关系基于行业经验总结。具体的具体数值和阈值需要 Step 2 中验证。

| 关系 ID | 关系名称 | 关系描述 | 关联对象 | 业务含义 | 行业通用性 |
|---------|---------|---------|---------|----------|----------|
| R001 | 成本超支与设计变更的关联 | 设计变更通常会导致成本超支，尤其是结构设计变更 | 成本、设计变更 | 设计变更次数较多时，成本超支风险显著增加 | 通用 |
| R002 | 利润率与收款周期的关系 | 收款周期过长会影响项目利润率 | 项目、资金流 | 收款周期延长，资金占用成本增加 | 通用 |
| R003 | 材料价格波动与成本的关系 | 主要材料价格波动显著影响成本 | 成本、材料价格 | 钢筋、混凝土等主要材料价格波动影响最大 | 通用 |
| R004 | 项目进度与成本支出的关系 | 项目进度滞后通常伴随成本超支 | 项目、成本 | 进度滞后时，成本超支概率增加 | 通用 |
| R005 | 质保金预留与资金流的关系 | 质保金通常为合同金额的一定比例，影响最终收款 | 项目、资金流 | 影响资金回笼时间 | 通用 |
| R006 | 管理费用与项目规模的关系 | 大型项目管理费用占比相对较低 | 项目、成本 | 规模效应降低单位管理成本 | 通用 |

### 4.2 指标间隐性关联关系

| 指标 A | 指标 B | 关联关系 | 关联强度 | 说明 |
|------|------|---------|---------|------|
| 成本偏差率 | 进度偏差率 | 正相关 | 强 | 成本超支通常伴随进度滞后 |
| 资金周转率 | 成本利润率 | 正相关 | 中 | 资金周转越快，成本利润率越高 |
| 变更签证率 | 成本超支率 | 正相关 | 强 | 变更签证越多，成本超支风险越大 |
| 单位面积成本 | 项目规模 | 负相关 | 中 | 项目规模越大，单位面积成本越低 |
| 合同履约率 | 成本利润率 | 正相关 | 中 | 合同履约率越高，成本利润率越高 |

### 5.1 行业独特规则清单

以下为房建行业特有的业务规则和操作模式，这些规则是行业经验的结晶，对项目成功至关重要。

| 诀窍 ID | 诀窍名称 | 规则描述 | 业务影响 | 行业通用性 |
|---------|---------|---------|---------|----------|
| T001 | 产值确认规则 | 产值需业主签字确认才能计入收入，否则只能作为待确产值 | 影响收入确认和利润计算 | 通用（房建行业） |

| T006 | 工程变更索赔规则 | 工程变更导致的成本增加可向业主索赔，但需提供充分的证据 | 影响成本控制和利润计算 | 通用（房建行业） |

### 5.2 行业操作模式

| 模式 ID | 模式名称 | 模式描述 | 优势 | 适用场景 |
|---------|---------|---------|------|----------|
| M001 | 总分包模式 | 将项目分为总承包和分包，总承包商负责整体管理，分包商负责具体施工 | 专业分工明确、风险分散 | 大型项目 |

| M004 | PPP模式 | 政府和社会资本合作模式，共同投资、建设、运营项目 | 减轻政府财政压力、提高项目效率 | 基础设施项目 |

## 六、L5：行业通用流程规则（新增）

### 6.0 行业通用流程规则总览

以下为建筑行业通用的业务流程和审批规则，这些是行业内通行的标准化流程。

| 规则 ID | 规则名称 | 核心流程 | 行业通用性 |
|--------|---------|---------|---------|
| PR001 | 产值确认流程 | 项目申报→监理审核→业主确认 | 通用 |
| PR002 | 成本报销流程 | 项目审核→公司审批 | 通用 |
| PR003 | 合同审批流程 | 商务谈判→法务审核→领导审批→签订盖章 | 通用 |
| PR004 | 资金支付流程 | 申请提交→审核确认→领导审批→财务支付 | 通用 |
| PR005 | 变更签证流程 | 变更申请→评估审核→三方确认→执行→结算 | 通用（房建行业） |
| PR006 | 工程验收流程 | 自检→初验→竣工验收→备案 | 通用（建筑行业） |

### 6.1 详细流程说明

#### PR001：产值确认流程
```
项目商务人员提交产值申报资料 → 监理单位审核工程量真实性 → 业主单位签字确认产值金额 → 财务部门计入收入
```

#### PR002：成本报销流程
```
项目人员提交报销申请 → 项目经理审核成本合理性 → 公司财务部门审核票据合规性 → 公司领导审批 → 财务部门支付款项
```

#### PR003：合同审批流程
```
商务部门进行合同谈判 → 法务部门审核合同条款 → 公司领导审批合同内容 → 双方签订盖章 → 合同归档
```

#### PR004：资金支付流程
```
项目人员提交资金支付申请 → 财务部门审核申请合理性 → 公司领导审批支付金额 → 财务部门执行支付 → 资金支付记录归档
```

#### PR005：变更签证流程
```
项目人员提交变更申请 → 技术部门评估变更影响 → 业主、监理、施工单位三方签字确认 → 项目执行变更 → 变更内容结算
```

#### PR006：工程验收流程
```
施工单位自检 → 监理单位初验 → 业主单位组织竣工验收 → 相关部门备案 → 交付使用
```
