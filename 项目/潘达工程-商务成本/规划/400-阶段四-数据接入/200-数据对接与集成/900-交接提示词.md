### 关键CLI命令
```bash
# 新建或拉取流程
dazi flow project pull --flow <id> --dir "<流程目录>"

# 提交节点代码
dazi flow node push --node <uuid> --dir "<流程目录>"

# 单节点测试
dazi flow run node-exec --node <uuid> --dir "<流程目录>"

# 查看变量
dazi flow variable pull --name <变量名> --dir "<流程目录>"

# 提交画布
dazi flow project push --dir "<流程目录>" --canvas
```

### 标准开发循环
1. 新建或拉取流程
2. 改代码（节点/code.sql 或 code.py）
3. 改画布配置（flow.json）
4. 提交节点代码（node push）
5. 单节点测试（node-exec）
6. 查看变量（variable pull）
7. 提交画布（project push）
8. 失败时查看错误（_run/*.last-error.md）
