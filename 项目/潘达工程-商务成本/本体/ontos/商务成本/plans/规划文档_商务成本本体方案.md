## 二、需求分析

### 2.1 业务背景

潘达建工（房建）商务成本管理场景，需构建**商务成本本体**，以业务语言描述「项目」「产值」「成本」「付款」「指标」「风险」及其关系，支撑：

- 项目产值与成本分析（已确认/待确认双轨制）
- 项目资金流分析（收款、付款、结余）
- 项目指标监控（毛利率、收款率、成本偏差率等）
- 项目风险预警（三色预警机制）
- 项目综合分析（多维度汇总、趋势分析）

### 2.2 核心业务问题

| 问题           | 说明                         |
| -------------- | ---------------------------- |
| 项目产值多少   | 已确认产值、待确认产值、总产值 |
| 项目成本多少   | 累计成本、本月成本、成本构成   |
| 资金流如何     | 收款、付款、结余、收现率、付现率 |
| 指标表现如何   | 毛利率、收款率、成本偏差率等   |
| 风险状况如何   | 风险类型、预警级别、风险描述   |
| 趋势如何       | 同比、环比、累计趋势           |

## 七、与参照示例的差异说明

### 7.1 业务域差异

| 对比项 | 产品销售示例 | 商务成本本体 | 差异说明 |
| ------ | ------------ | ------------ | -------- |
| **业务域** | 销售、订单、产品 | 项目、产值、成本、付款、指标、风险 | 完全不同的业务域 |
| **核心对象** | Product, SalesOrder, Customer | Project, ProjectOutput, ProjectCost, ProjectPayment, ProjectIndicator, ProjectRisk | 项目管理vs销售管理 |
| **时间维度** | 订单日期 | 报告期间（月度） | 统计周期不同 |
| **业务规则** | 无特殊规则 | ER001-ER006企业特有规则 | 企业特有规则是核心差异 |

### 7.2 物理表差异

| 对比项 | 产品销售示例 | 商务成本本体 | 差异说明 |
| ------ | ------------ | ------------ | -------- |
| **维度表数量** | 4个（dim_date, dim_product, dim_customer, dim_channel） | 9个（dim_date, dim_region, dim_department, dim_owner, dim_cost_subject, dim_project, dim_company, dim_supplier, dim_contract） | 维度表更多，反映项目管理复杂性 |
| **事实表数量** | 1个（fact_sales_order_line） | 6个（fact_project_output, fact_project_cost, fact_project_indicator, fact_project_payment, fact_project_balance, fact_project_risk） | 事实表更多，反映多维度分析需求 |
| **表间关系** | 5条关系 | 16条关系 | 关系更复杂 |

### 7.3 Cube差异

| 对比项 | 产品销售示例 | 商务成本本体 | 差异说明 |
| ------ | ------------ | ------------ | -------- |
| **Cube数量** | 4个 | 15个 | Cube数量更多 |
| **Cube类别** | Process, Subject | Time, Subject, Process | 增加Time类别 |

### 7.4 本体层差异

| 对比项 | 产品销售示例 | 商务成本本体 | 差异说明 |
| ------ | ------------ | ------------ | -------- |
| **Object Type数量** | 5个 | 15个 | 对象类型更多 |
| **Link Type数量** | 8种 | 12种 | 链接类型更多 |
| **企业规则实现** | 无 | ER001-ER006 | 企业特有规则是核心差异 |

### 7.5 函数差异

| 对比项 | 产品销售示例 | 商务成本本体 | 差异说明 |
| ------ | ------------ | ------------ | -------- |
| **函数数量** | 7个 | 7个 | 数量相近 |

| **分析维度** | 产品、客户、渠道 | 项目、产值、成本、付款、指标、风险 | 分析维度完全不同 |
