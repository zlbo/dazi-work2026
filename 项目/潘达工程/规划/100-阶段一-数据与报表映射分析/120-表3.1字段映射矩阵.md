# 120-表3.1字段映射矩阵

## 一、映射关系概述

表3.1的数据主要来源于:
1. **手动填写**(70%的数据)
2. **公式计算**(25%的数据)
3. **引用表3.1.1**(5%的数据)

表3.1是业主产值数据的**源头表**,数据直接录入后供表3引用。

## 九、SQL查询示例

### 9.1 查询项目产值数据

```sql
SELECT 
    code as 产值类型,
    output_confirmed as 已确认产值,
    output_unconfirmed as 待确认产值,
    (output_confirmed + output_unconfirmed) as 合计产值,
    output_date as 核定日期,
    output_period as 核定期间
FROM tb_project_output
WHERE project_id = ? 
  AND year = ? 
  AND month = ?
ORDER BY code;
```

### 9.2 查询应收款数据

```sql
SELECT 
    receivable_amount as 应收款金额,
    receivable_formula as 计算公式
FROM tb_project_receivable
WHERE project_id = ? 
  AND year = ? 
  AND month = ?;
```

### 9.3 查询待确产值分析

```sql
SELECT 
    code as 编号,
    description as 描述,
    contract_summary as 合同摘要,
    reason as 待确原因,
    amount as 待确金额
FROM tb_project_output_unconfirmed
WHERE project_id = ? 
  AND year = ? 
  AND month = ?
ORDER BY code;
```

## 十、数据同步方案

### 10.1 同步流程

```
1. 手动录入表3.1数据
    ↓
2. 公式自动计算
    ↓
3. 保存到数据库

    ↓
```
