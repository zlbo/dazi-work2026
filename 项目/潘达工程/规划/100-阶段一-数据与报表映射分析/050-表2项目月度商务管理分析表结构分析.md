## 二、报表整体结构

表2分为**五大板块**:

```
┌────────────────────────────────────────────────────┐
│ 表头:项目名称、报告期间、单位                        │
├────────────────────────────────────────────────────┤
│ 一、项目总体评价(行3-6)                             │
│   - 管理费:上缴费率、预计应缴、已缴、欠缴            │
│   - 利润:开累利润额、利润率(已确+待确、已确)         │
│   - 预警:红黄绿灯状态及原因分析                      │
├────────────────────────────────────────────────────┤
│ 二、工程概况(行7-15)                                │
│   - 基本信息:工程名称、建设单位、地点、规模          │
│   - 时间信息:合同开工/竣工、实际开工/竣工            │
│   - 合同信息:合同价款、单方造价、支付方式            │
│   - 目标信息:预估结算金额、本年度目标产值            │
├────────────────────────────────────────────────────┤
│ 三、工程形象进度简要概况(行16-17)                   │
│   - 文本描述区域                                    │
├────────────────────────────────────────────────────┤
│ 四、各项商务数据风险清单状况(行18-36)               │
│   - 进度风险:产值完成额、完成率(开累、本年度)        │
│   - 利润风险:利润额、利润率(多种计算方式)            │
│   - 造价管理风险:预算、签证、认价、结算、目标成本    │
├────────────────────────────────────────────────────┤
│ 五、主控费用状况(行37-47)                           │
│   - 周转材费用、垂直运输费用                         │
│   - 土建劳务费用、安全文明费用                       │

│   - 每项包含:目标成本、合同金额、实际发生            │
└────────────────────────────────────────────────────┘
```

### 4.3 计算逻辑分析

| 计算项 | 计算公式 | 说明 |
|-------|---------|------|
| 预计应缴管理费 | 管理费率 × 预估结算金额 | 基于预估完工金额计算 |
| 欠公司管理费 | 合同额 × 管理费率 - 已缴管理费 | 应缴与已缴的差额 |
| 单方造价 | 合同价款 / 建筑面积 | 每平米造价指标 |
| 开累产值完成率 | 开累产值完成额 / 预估结算金额 | 产值完成进度 |
| 本年度产值完成率 | 本年度产值完成额 / 本年度目标产值 | 年度产值完成进度 |
| 利润额(已确-待确成本) | 已确产值 - (已确成本 + 待确成本) | 考虑待确成本的利润额 |
| 利润率(已确-待确成本) | [已确产值-(已确成本+待确成本)] / 已确产值 | 考虑待确成本的利润率 |
| 主控费用单方造价 | 费用金额 / 建筑面积 | 各项费用的单方指标 |

## 五、业务逻辑分析

### 5.1 项目总体评价逻辑

**管理费评价**:
- 上缴管理费费率:从表3.2获取,反映项目管理费缴纳比例
- 预计完工应缴纳管理费:根据预估完工结算金额计算应缴总额
- 已缴纳管理费:从表3获取实际已缴金额
- 欠公司管理费:应缴与已缴的差额,反映管理费缴纳情况

**利润评价**:
- 提供两种利润视角:
  1. 已确+待确:反映项目整体盈利情况
  2. 已确:反映已确认部分的盈利情况
- 利润额和利润率同时展示,全面评价盈利能力

**预警机制**:
- 红黄绿灯三色预警
- 绿灯:指标正常,利润率为正
- 黄灯:2-5项指标超出参考范围,或利润率连续2个月为负
- 红灯:≥6项指标超出参考范围,或利润率连续≥3个月为负

### 5.2 风险清单逻辑

**进度风险**:
- [110]开累产值完成额:累计完成产值绝对值
- [120]开累产值完成率:与预估结算金额对比,判断进度是否滞后
- [130]本年度产值完成额:本年度完成产值绝对值
- [140]本年度产值完成率:与本年度目标产值对比,判断年度进度

**利润风险**:
- [210]开累至今利润额(已确+待确):整体利润水平
- [220]利润率(已确+待确):整体盈利能力
- [223]利润额(已确产值-已确成本-待确成本):考虑待确成本影响的利润
- [225]利润率(已确产值-已确成本-待确成本)/已确产值:考虑待确成本的盈利能力
- [230]开累至今利润额(已确):已确认部分的利润水平
- [240]利润率(已确):已确认部分的盈利能力

**造价管理风险**:
- [310]施工图预算:是否按时办理,是否存在争议
- [320]签证、索赔办理情况:办理进度和完成率
- [330]认质认价办理情况:办理进度和完成率
- [340]业主结算:是否按时办理,是否存在争议
- [350]目标成本:是否完成,是否在目标范围内
- [360]完工成本预分析:是否完成,是否达到利润目标

### 5.3 主控费用逻辑

**费用管控思路**:
- 每项主控费用从三个维度对比:
  1. 目标成本:计划控制目标
  2. 合同金额:已签订合同总额
  3. 实际发生:实际支出情况

**单方造价指标**:
- 所有金额指标均计算单方造价(元/m²)
- 便于不同规模项目的横向对比
- 建筑面积统一使用J10单元格(从表3引用)

**费用范围说明**:
- 明确各项费用的包含范围
- 避免重复计算或遗漏
- 特殊情况在备注中说明

### 6.2 手动填写字段

以下字段需要手动填写:

- 合同信息:D14、J14(支付方式、资金来源)

- 主控费用分析:L列(存在问题及改进措施)

## 七、报表特点总结

### 7.1 优势特点

1. **全面性**:涵盖项目评价、工程概况、风险清单、主控费用四大维度
2. **层次性**:从总体评价到具体费用,层层递进
3. **预警性**:红黄绿灯机制,直观展示风险状态
4. **可比性**:单方造价指标,便于横向对比
5. **可追溯性**:数据主要来源于表3,确保数据一致性

### 7.2 数据依赖

- **强依赖表3**:78%的数据来源于表3,表3数据质量直接影响表2
- **弱依赖表3.2**:仅引用管理费率,影响较小
- **手动填写较多**:工程概况、风险分析、费用金额等需手动填写

### 7.3 应用场景

- **月度商务管理会议**:展示项目整体运营状况
- **风险预警**:及时发现进度、利润、造价管理风险
- **费用管控**:监控主控费用执行情况
- **决策支持**:为管理层提供项目评价依据
