## 快速启动

### 1. 环境准备
```powershell
# 登录平台
dazi auth login

# 确认空间
dazi space list
```

### 2. 发布本体脚本
```powershell
# 发布初始化脚本
dazi onto script publish 项目/潘达工程-商务成本/脚本/setup/ontology_init.py --space space__panda_cost01 --type setup

# 发布灌数脚本
dazi onto script publish 项目/潘达工程-商务成本/脚本/setup/seed_data.py --space space__panda_cost01 --type data
```

### 3. 验证本体
```powershell
# 查看Object Type
dazi onto object-type list --space space__panda_cost01

# 查看Cube
dazi onto cube list --space space__panda_cost01
```
