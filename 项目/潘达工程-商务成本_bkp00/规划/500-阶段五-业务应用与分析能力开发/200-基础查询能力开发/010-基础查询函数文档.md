## 一、函数概述

### 1.1 设计目标

基于本体设计和数据对接成果，开发基础查询函数，支持AI助手进行自然语言问答，实现浅层次的指标查询能力。

### 1.2 函数分类

| 分类 | 说明 | 示例 |
|------|------|------|
| **指标查询函数** | 查询单个或多个指标值 | getIndicatorValue |
| **时间分析函数** | 同比、环比、累计计算 | getYoYGrowth |
| **统计分析函数** | 平均值、求和、排名 | getAverage |
| **维度筛选函数** | 按维度筛选数据 | filterByProject |

### 1.3 函数命名规范

```
{action}{ObjectType}{Property}{Modifier}

- action: get/filter/calculate
- ObjectType: Project/Output/Cost/Payment/Balance/Indicator/Risk
- Property: Value/Ratio/Growth/Total
- Modifier: YoY/QoQ/Accumulated
```

## 二、指标查询函数

### 2.1 getProjectIndicatorValue

**功能**：查询项目指标值

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectId | STRING | 是 | 项目ID |
| indicatorCode | STRING | 是 | 指标编码 |
| reportPeriod | STRING | 否 | 报告期间（默认当前） |

**返回值**：

| 字段 | 类型 | 说明 |
|------|------|------|
| projectId | STRING | 项目ID |
| projectName | STRING | 项目名称 |
| indicatorCode | STRING | 指标编码 |
| indicatorName | STRING | 指标名称 |
| indicatorValue | DECIMAL | 指标值 |
| unit | STRING | 单位 |
| reportPeriod | STRING | 报告期间 |

**示例调用**：
```
getProjectIndicatorValue("P001", "profit_rate", "2024-01")
```

**AI问答映射**：
- "P001项目2024年1月的利润率是多少？"

---

### 2.2 getMultipleIndicators

**功能**：批量查询多个指标

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectId | STRING | 是 | 项目ID |
| indicatorCodes | LIST | 是 | 指标编码列表 |
| reportPeriod | STRING | 否 | 报告期间 |

**返回值**：指标值列表

**示例调用**：
```
getMultipleIndicators("P001", ["profit_rate", "cost_ratio", "collection_rate"], "2024-01")
```

**AI问答映射**：
- "P001项目的利润率、成本率、回款率分别是多少？"

## 三、时间分析函数

### 3.1 getYoYGrowth

**功能**：计算同比增长率

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectId | STRING | 是 | 项目ID |
| indicatorCode | STRING | 是 | 指标编码 |
| reportPeriod | STRING | 否 | 报告期间 |

**返回值**：

| 字段 | 类型 | 说明 |
|------|------|------|
| currentValue | DECIMAL | 当前值 |

| growthRate | DECIMAL | 同比增长率(%) |

**示例调用**：
```
getYoYGrowth("P001", "output_total", "2024-01")
```

- "P001项目2024年1月产值同比增长了多少？"

## 四、统计分析函数

### 4.1 getAverageValue

**功能**：计算平均值

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectIds | LIST | 否 | 项目ID列表（默认全部） |
| indicatorCode | STRING | 是 | 指标编码 |
| reportPeriod | STRING | 否 | 报告期间 |

**返回值**：

| 字段 | 类型 | 说明 |
|------|------|------|
| averageValue | DECIMAL | 平均值 |
| count | INTEGER | 项目数 |
| minValue | DECIMAL | 最小值 |
| maxValue | DECIMAL | 最大值 |

**示例调用**：
```
getAverageValue(["P001", "P002", "P003"], "profit_rate", "2024-01")
```

**AI问答映射**：
- "所有项目的平均利润率是多少？"

---

### 4.2 getRanking

**功能**：获取排名

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| indicatorCode | STRING | 是 | 指标编码 |
| reportPeriod | STRING | 否 | 报告期间 |
| topN | INTEGER | 否 | 取前N名（默认10） |

**返回值**：排名列表（项目ID、项目名称、指标值、排名）

**示例调用**：
```
getRanking("profit_rate", "2024-01", 5)
```

**AI问答映射**：
- "利润率排名前五的项目是哪些？"

## 五、维度筛选函数

### 5.1 filterByCompany

**功能**：按分公司筛选

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| companyId | STRING | 是 | 分公司ID |
| indicatorCode | STRING | 是 | 指标编码 |
| reportPeriod | STRING | 否 | 报告期间 |

**返回值**：该分公司下所有项目的指标值列表

**示例调用**：
```
filterByCompany("C001", "profit_rate", "2024-01")
```

**AI问答映射**：
- "第一分公司所有项目的利润率是多少？"

---

### 5.2 filterByPeriod

**功能**：按期间范围筛选

**参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| projectId | STRING | 是 | 项目ID |
| indicatorCode | STRING | 是 | 指标编码 |
| startPeriod | STRING | 是 | 开始期间 |
| endPeriod | STRING | 是 | 结束期间 |

**返回值**：期间范围内的指标值列表

**示例调用**：
```
filterByPeriod("P001", "output_total", "2024-01", "2024-12")
```

**AI问答映射**：
- "P001项目2024年各月产值是多少？"
