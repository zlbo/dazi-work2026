# 240-项目收支汇总表字段映射矩阵

## 一、映射关系概述

项目收支汇总表的数据主要来源于:
1. **手动填写**(80%的数据)
2. **公式计算**(20%的数据)

项目收支汇总表是**核心数据源表**，为表1、表2、表3提供资金和收款数据。

## 六、SQL查询示例

### 6.1 查询资金余额

```sql
SELECT 
    code as 代码,
    subject_name as 科目名称,
    project_amount as 项目金额,
    company_amount as 公司金额,
    total_amount as 合计
FROM tb_project_balance_detail
WHERE project_id = ? 
  AND balance_id = ?
  AND code LIKE '1%'
ORDER BY code;
```

### 6.2 查询收款金额

```sql
SELECT 
    code as 代码,
    subject_name as 科目名称,
    project_amount as 项目金额,
    company_amount as 公司金额,
    total_amount as 合计
FROM tb_project_balance_detail
WHERE project_id = ? 
  AND balance_id = ?
  AND code LIKE '2%'
ORDER BY code;
```

### 6.3 查询收支汇总

```sql
SELECT 
    SUM(CASE WHEN code LIKE '1%' THEN total_amount ELSE 0 END) as 资金余额合计,
    SUM(CASE WHEN code LIKE '2%' THEN total_amount ELSE 0 END) as 收款金额合计,
    SUM(total_amount) as 总合计
FROM tb_project_balance_detail
WHERE project_id = ? 
  AND balance_id = ?;
```
