# 建筑工程行业商务成本管理核心业务流程清单

## 一、产值确认流程

【P001】产值确认流程
- 流程目标：确认项目已完成工程量对应的产值金额
- 适用场景：项目进度达到合同约定的节点时
- 流程步骤：
  1. 项目申报：
     - 执行主体：项目商务人员
     - 输入：工程进度报表、工程量清单、质量验收记录
     - 输出：产值申报单
     - 审批要求：否
     - 时间要求：1个工作日
  2. 部门审核：
     - 执行主体：商务部门负责人
     - 输入：产值申报单、进度报表
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  3. 业主确认：
     - 执行主体：业主代表或监理
     - 输入：审核后的产值申报单、工程量确认单
     - 输出：业主签字确认单
     - 审批要求：是
     - 时间要求：7个工作日
  4. 财务入账：
     - 执行主体：财务人员
     - 输入：业主确认单
     - 输出：应收账款入账凭证
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：项目申报→部门审核→业主确认→财务入账
- 异常处理：业主对工程量有异议时，需重新核对工程量；审核不通过时，退回修改
- 平台映射：Action Type（产值确认流程）

## 二、成本报销流程

【P002】成本报销流程
- 流程目标：完成费用报销，支付款项
- 适用场景：员工发生业务费用后
- 流程步骤：
  1. 费用申请：
     - 执行主体：员工
     - 输入：费用发票、费用明细、业务说明
     - 输出：费用报销单
     - 审批要求：否
     - 时间要求：1个工作日
  2. 部门审核：
     - 执行主体：部门负责人
     - 输入：费用报销单、发票
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  3. 财务审核：
     - 执行主体：财务人员
     - 输入：费用报销单、发票、合同（如有）
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  4. 领导审批：
     - 执行主体：分管领导
     - 输入：费用报销单、审核意见
     - 输出：审批意见
     - 审批要求：是（金额>5000元）
     - 时间要求：3个工作日
  5. 资金支付：
     - 执行主体：财务人员
     - 输入：审批通过的报销单
     - 输出：支付凭证
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：费用申请→部门审核→财务审核→领导审批（如需）→资金支付
- 异常处理：发票不合规时，退回重新提供；金额异常时，需提供额外说明
- 平台映射：Action Type（成本报销流程）

## 三、合同审批流程

【P003】合同审批流程
- 流程目标：完成合同签订前的审批
- 适用场景：商务谈判完成，准备签订合同
- 流程步骤：
  1. 合同起草：
     - 执行主体：商务人员
     - 输入：谈判纪要、合同草案
     - 输出：合同初稿
     - 审批要求：否
     - 时间要求：2个工作日
  2. 部门审核：
     - 执行主体：商务部门负责人
     - 输入：合同初稿、谈判纪要
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：3个工作日
  3. 法务审核：
     - 执行主体：法务人员
     - 输入：合同初稿、审核意见
     - 输出：法律意见
     - 审批要求：是
     - 时间要求：3个工作日
  4. 财务审核：
     - 执行主体：财务人员
     - 输入：合同初稿、法律意见
     - 输出：财务意见
     - 审批要求：是
     - 时间要求：2个工作日
  5. 领导审批：
     - 执行主体：总经理或授权领导
     - 输入：合同初稿、各部门意见
     - 输出：审批意见
     - 审批要求：是
     - 时间要求：3个工作日
  6. 合同签订：
     - 执行主体：商务人员
     - 输入：审批通过的合同
     - 输出：正式合同文本
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：合同起草→部门审核→法务审核→财务审核→领导审批→合同签订
- 异常处理：条款有风险时，退回修改；金额较大时，需董事会审批
- 平台映射：Action Type（合同审批流程）

## 四、资金支付流程

【P004】资金支付流程
- 流程目标：完成对外款项支付
- 适用场景：需要向供应商或分包商支付款项时
- 流程步骤：
  1. 支付申请：
     - 执行主体：项目商务人员
     - 输入：合同、发票、验收证明、付款申请单
     - 输出：付款申请
     - 审批要求：否
     - 时间要求：1个工作日
  2. 部门审核：
     - 执行主体：商务部门负责人
     - 输入：付款申请、合同、发票
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  3. 财务审核：
     - 执行主体：财务人员
     - 输入：付款申请、合同、发票、资金计划
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  4. 领导审批：
     - 执行主体：分管领导
     - 输入：付款申请、审核意见
     - 输出：审批意见
     - 审批要求：是（金额>10万元）
     - 时间要求：2个工作日
  5. 资金支付：
     - 执行主体：财务人员
     - 输入：审批通过的付款申请
     - 输出：支付凭证
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：支付申请→部门审核→财务审核→领导审批（如需）→资金支付
- 异常处理：发票不合规时，退回重新提供；超合同约定付款时，需特殊审批
- 平台映射：Action Type（资金支付流程）

## 五、变更签证流程

【P005】变更签证流程
- 流程目标：完成设计变更或现场签证的审批和确认
- 适用场景：项目实施过程中发生设计变更或现场签证时
- 流程步骤：
  1. 变更申请：
     - 执行主体：项目工程师
     - 输入：变更原因、变更方案、工程量计算书
     - 输出：变更签证申请单
     - 审批要求：否
     - 时间要求：1个工作日
  2. 技术审核：
     - 执行主体：技术部门负责人
     - 输入：变更签证申请单、设计图纸
     - 输出：技术审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  3. 商务审核：
     - 执行主体：商务人员
     - 输入：变更签证申请单、技术审核意见、费用估算
     - 输出：商务审核意见
     - 审批要求：是
     - 时间要求：2个工作日
  4. 业主确认：
     - 执行主体：业主代表或监理
     - 输入：变更签证申请单、审核意见
     - 输出：业主确认单
     - 审批要求：是
     - 时间要求：7个工作日
  5. 台账记录：
     - 执行主体：商务人员
     - 输入：业主确认单
     - 输出：变更签证台账
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：变更申请→技术审核→商务审核→业主确认→台账记录
- 异常处理：费用估算不合理时，退回重新核算；业主不确认时，需进一步沟通
- 平台映射：Action Type（变更签证流程）

## 六、竣工结算流程

【P006】竣工结算流程
- 流程目标：完成项目最终结算
- 适用场景：项目竣工验收合格后
- 流程步骤：
  1. 结算准备：
     - 执行主体：项目商务人员
     - 输入：合同、变更签证、工程量计算书、材料价差调整资料
     - 输出：结算资料清单
     - 审批要求：否
     - 时间要求：5个工作日
  2. 结算编制：
     - 执行主体：商务人员
     - 输入：结算资料、合同条款
     - 输出：竣工结算书
     - 审批要求：否
     - 时间要求：10个工作日
  3. 内部审核：
     - 执行主体：商务部门负责人、财务人员
     - 输入：竣工结算书、结算资料
     - 输出：内部审核意见
     - 审批要求：是
     - 时间要求：5个工作日
  4. 业主审核：
     - 执行主体：业主或审计机构
     - 输入：竣工结算书、结算资料
     - 输出：审核意见或审计报告
     - 审批要求：是
     - 时间要求：30个工作日
  5. 争议解决：
     - 执行主体：商务人员
     - 输入：审核意见、争议事项
     - 输出：争议解决方案
     - 审批要求：是
     - 时间要求：视情况而定
  6. 最终确认：
     - 执行主体：双方负责人
     - 输入：结算书、审核意见
     - 输出：结算确认书
     - 审批要求：是
     - 时间要求：3个工作日
- 审批节点：结算准备→结算编制→内部审核→业主审核→争议解决→最终确认
- 异常处理：审核争议时，组织谈判或申请仲裁；资料不全时，补充提供
- 平台映射：Action Type（竣工结算流程）

## 七、库存盘点流程

【P007】库存盘点流程
- 流程目标：定期盘点项目材料库存，确保账实相符
- 适用场景：每月末或项目阶段性收尾时
- 流程步骤：
  1. 盘点准备：
     - 执行主体：材料管理员
     - 输入：库存台账、盘点表模板
     - 输出：盘点计划
     - 审批要求：否
     - 时间要求：1个工作日
  2. 实地盘点：
     - 执行主体：材料管理员、财务人员
     - 输入：盘点计划、库存台账
     - 输出：盘点记录表
     - 审批要求：否
     - 时间要求：2个工作日
  3. 差异分析：
     - 执行主体：材料管理员
     - 输入：盘点记录表、库存台账
     - 输出：差异分析报告
     - 审批要求：否
     - 时间要求：1个工作日
  4. 审批确认：
     - 执行主体：物资部门负责人
     - 输入：差异分析报告
     - 输出：审批意见
     - 审批要求：是
     - 时间要求：1个工作日
  5. 台账调整：
     - 执行主体：材料管理员
     - 输入：审批意见、差异分析报告
     - 输出：调整后的库存台账
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：盘点准备→实地盘点→差异分析→审批确认→台账调整
- 异常处理：差异较大时，重新盘点；盘亏需追究责任时，启动调查
- 平台映射：Action Type（库存盘点流程）

## 八、材料调拨流程

【P008】材料调拨流程
- 流程目标：实现项目间或部门间材料转移
- 适用场景：需要从一个项目调拨材料到另一个项目时
- 流程步骤：
  1. 调拨申请：
     - 执行主体：调入项目材料员
     - 输入：材料需求清单、调出项目库存情况
     - 输出：材料调拨申请单
     - 审批要求：否
     - 时间要求：1个工作日
  2. 调出审核：
     - 执行主体：调出项目负责人
     - 输入：调拨申请单、调出项目库存台账
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：1个工作日
  3. 调入审核：
     - 执行主体：调入项目负责人
     - 输入：调拨申请单、审核意见
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：1个工作日
  4. 物资部门审批：
     - 执行主体：物资部门负责人
     - 输入：调拨申请单、双方审核意见
     - 输出：审批意见
     - 审批要求：是（金额>5000元）
     - 时间要求：1个工作日
  5. 执行调拨：
     - 执行主体：材料管理员
     - 输入：审批通过的调拨申请单
     - 输出：调拨单、库存变更记录
     - 审批要求：否
     - 时间要求：2个工作日
- 审批节点：调拨申请→调出审核→调入审核→物资部门审批（如需）→执行调拨
- 异常处理：调出项目库存不足时，调整调拨数量；跨项目调拨需考虑运输成本
- 平台映射：Action Type（材料调拨流程）

## 九、废旧物资处理流程

【P009】废旧物资处理流程
- 流程目标：规范处理项目废旧物资，实现资源回收
- 适用场景：项目产生废旧物资需要处置时
- 流程步骤：
  1. 物资鉴定：
     - 执行主体：项目工程师、材料管理员
     - 输入：废旧物资清单、物资状况说明
     - 输出：废旧物资鉴定报告
     - 审批要求：否
     - 时间要求：1个工作日
  2. 价值评估：
     - 执行主体：财务人员、物资部门
     - 输入：废旧物资鉴定报告、市场行情
     - 输出：价值评估报告
     - 审批要求：否
     - 时间要求：1个工作日
  3. 处理方案审批：
     - 执行主体：物资部门负责人、财务负责人
     - 输入：鉴定报告、评估报告、处理方案（拍卖/回收/报废）
     - 输出：审批意见
     - 审批要求：是
     - 时间要求：2个工作日
  4. 公开处置：
     - 执行主体：物资部门
     - 输入：审批通过的处理方案
     - 输出：处置记录、回收款项
     - 审批要求：否
     - 时间要求：视情况而定
  5. 财务入账：
     - 执行主体：财务人员
     - 输入：处置记录、回收款项凭证
     - 输出：入账凭证
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：物资鉴定→价值评估→处理方案审批→公开处置→财务入账
- 异常处理：处置价格明显偏低时，重新评估；重大废旧物资需公开招标
- 平台映射：Action Type（废旧物资处理流程）

## 十、机械费计算流程

【P010】机械费计算流程
- 流程目标：准确计算施工机械使用费用
- 适用场景：每月末机械费用结算时
- 流程步骤：
  1. 数据收集：
     - 执行主体：机械管理员
     - 输入：机械台班记录、油料消耗记录、维修保养记录
     - 输出：机械使用明细表
     - 审批要求：否
     - 时间要求：2个工作日
  2. 费用计算：
     - 执行主体：商务人员
     - 输入：机械使用明细表、台班单价、油料单价
     - 输出：机械费计算表
     - 审批要求：否
     - 时间要求：2个工作日
  3. 审核确认：
     - 执行主体：商务部门负责人
     - 输入：机械费计算表、机械使用明细表
     - 输出：审核意见
     - 审批要求：是
     - 时间要求：1个工作日
  4. 财务入账：
     - 执行主体：财务人员
     - 输入：审核通过的机械费计算表
     - 输出：入账凭证
     - 审批要求：否
     - 时间要求：1个工作日
- 审批节点：数据收集→费用计算→审核确认→财务入账
- 异常处理：台班记录与实际不符时，核实后调整；费用异常时，分析原因
- 平台映射：Action Type（机械费计算流程）
