# 建筑工程行业商务成本管理显性业务关系清单

## 一、包含关系

【R001】项目包含合同

- 业务含义：一个项目可以签订多个合同（总包合同、分包合同、采购合同等）
- 行业特点：通用
- 映射建议：Link Type（包含关系）

## 四、计算关系

【R017】利润计算
- 关系类型：计算
- 源实体：产值
- 目标实体：成本
- 基数：1:1
- 业务含义：利润 = 产值 - 成本
- 行业特点：通用
- 映射建议：派生属性（利润）

【R018】成本偏差计算
- 关系类型：计算
- 源实体：成本
- 目标实体：预算
- 基数：1:1
- 业务含义：成本偏差 = 实际成本 - 预算成本
- 行业特点：通用
- 映射建议：派生属性（成本偏差）

【R019】成本偏差率计算
- 关系类型：计算
- 源实体：成本偏差
- 目标实体：预算
- 基数：1:1
- 业务含义：成本偏差率 = 成本偏差 / 预算成本 × 100%
- 行业特点：通用
- 映射建议：派生属性（成本偏差率）

【R020】已付比例计算
- 关系类型：计算
- 源实体：付款
- 目标实体：合同
- 基数：N:1
- 业务含义：已付比例 = 已付款金额 / 合同金额 × 100%
- 行业特点：通用
- 映射建议：派生属性（已付比例）

【R021】应收金额计算
- 关系类型：计算
- 源实体：产值
- 目标实体：付款
- 基数：1:N
- 业务含义：应收金额 = 确认产值 - 已收款金额
- 行业特点：通用
- 映射建议：派生属性（应收金额）
