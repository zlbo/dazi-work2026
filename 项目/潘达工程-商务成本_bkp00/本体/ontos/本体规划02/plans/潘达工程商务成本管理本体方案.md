### 1.1 业务背景

潘达工程为**房建施工项目**的商务成本智能决策场景。项目月度需汇总**产值确权、成本付款、资金收支、核心指标与风险预警**（对应 Excel 表 1–3 及数据库现有表），支撑：

- **商务总览**：项目数、产值、成本、利润率、确权率、回款率
- **产值分析**：已确认 / 待确认产值双轨、确权率、同比环比
- **成本分析**：已确认 / 待确认成本、人工 / 材料 / 设备结构、成本刚性度
- **付款与资金**：应付款、已付款、付款进度、资金结余因素分解
- **收支科目**：项目层 / 公司层收支汇总、科目结构
- **指标与风险**：7 项核心指标达成、三色预警、风险清单
- **组织对标**：分公司 / 项目间横向对比

### 1.2 核心业务问题

| 问题 | 说明 |
| ---- | ---- |
| 赚不赚钱 | 项目毛利率、成本率、利润率趋势 |
| 产值确没确 | 已确认 vs 待确认产值、确权率 |
| 成本控没控 | 目标成本 vs 已发生、待确成本、偏差率 |
| 钱收没收 | 回款率、收现率、应收清欠 |
| 钱付没付 | 应付款、付款率、未付款余额 |
| 资金够不够 | 资金结余 = 收款 − 付款，因素分解 |
| 风险大不大 | 指标预警、风险清单、健康度 |
| 哪个项目 / 分公司是短板 | Top 风险项目、分公司对标 |

### 1.3 与参照示例的差异说明

| 维度 | 销售示例 | 利润 / GL 示例 | 设备运营示例 | **本方案（潘达商务成本）** |
| ---- | -------- | -------------- | ------------ | -------------------------- |
| 核心锚点 | 订单 / 客户 / 产品 | 科目 / 分录 / 预算 | 设备 / 停机 / OEE | **项目 / 产值 / 成本 / 付款 / 收支** |
| 主事实粒度 | 订单行 | 凭证行 | 设备日汇总 + 停机事件 | **项目 × 报告期间（月度）** |
| 核心 KPI | 销售额、渠道占比 | 收入、成本、利润率 | OEE、可用率、MTBF | **毛利率、确权率、成本率、回款率、资金结余** |
| 组织维度 | 区域 / 渠道 | 成本中心 | 厂区 → 单元 → 设备 | **大区 → 分公司 → 项目 → 标段**；**业主 / 主责部门** |
| 对比分析 | 同比销售 | 预实对比 | 计划 vs 实际运行 | **已确 vs 待确、目标 vs 实际成本** |
| 行业特征 | 通用零售 | 制造财务 GL | 化工连续生产 | **房建施工、月度商务台账、双轨确权** |
| 函数设计 | 销售结构 / Top 产品 | 科目 / 利润项结构 | OEE / 停机 / 维保 | **产值 / 成本 / 付款 / 风险专项** |

| output_record_count | output_id | count | 记录数 |

| 度量名 | 表达式（示意） | 说明 | 对应 030 属性 |
