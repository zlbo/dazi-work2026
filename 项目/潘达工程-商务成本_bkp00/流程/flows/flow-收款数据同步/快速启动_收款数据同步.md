# 收款数据同步 - 快速启动指南

## 流程信息

| 属性 | 值 |
|------|-----|
| **流程名** | 收款数据同步 |
| **流程ID** | 140 |
| **数据源** | `tb_project_cost_receipt_detail` |
| **目标表** | `fact_project_receipt` |
| **数据空间** | `space__panda_construction` |

## 节点链

```mermaid
start --> 读取收款数据 --> 数据转换 --> 质量检查 --> 写入收款数据 --> end
```

## 变量传递

| 变量名 | 节点 | 说明 |
|--------|------|------|
| `receipt_raw_data` | 读取收款数据 | 源数据原始输出 |
| `project_receipt_data` | 数据转换 | 转换后数据 |
| `validated_receipt_data` | 质量检查 | 校验后数据 |

## 运行命令

```powershell
# 进入流程目录
cd "d:\GitHub\dazi-work\项目\潘达工程-商务成本\流程\flows\flow-收款数据同步"

# 修复元数据
dazi flow project repair-meta --dir .

# 提交画布
dazi flow project push --dir . --canvas

# 提交节点代码
dazi flow node push --node d96222ce-aaa9-4050-b782-830c2f2de9ee --dir .
dazi flow node push --node e5f8a3b9-2c7d-4e8a-9f6b-1a2c3d4e5f6d --dir .
dazi flow node push --node f7c4e8d2-1a3b-5c6d-7e8f-9a0b1c2d3e4i --dir .

# 执行流程
dazi flow run node-exec --node d96222ce-aaa9-4050-b782-830c2f2de9ee --dir .
dazi flow run node-exec --node e5f8a3b9-2c7d-4e8a-9f6b-1a2c3d4e5f6d --dir .
dazi flow run node-exec --node f7c4e8d2-1a3b-5c6d-7e8f-9a0b1c2d3e4i --dir .

# 拉取变量验证
dazi flow variable pull --name receipt_raw_data --dir .
dazi flow variable pull --name validated_receipt_data --dir .
```

## 字段映射

| 源字段 | 目标字段 | 说明 |
|--------|----------|------|
| `id` | `receipt_id` | 收款记录ID |
| `project_id` | `project_id` | 项目ID |
| `contract_id` | `contract_id` | 合同ID |
| `year`/`month` | `date_key` | 日期键(YYYYMMDD) |
| `year`/`month` | `report_period` | 报告期(YYYY-MM) |
| `base_contract_code` | `contract_code` | 合同编码 |
| `base_contract_name` | `contract_name` | 合同名称 |
| `base_contract_amount` | `contract_amount` | 合同金额 |
| `receipt_amount_acc` | `receipt_amount` | 累计收款金额 |
| `receipt_ratio_acc` | `receipt_ratio` | 收款比例 |
