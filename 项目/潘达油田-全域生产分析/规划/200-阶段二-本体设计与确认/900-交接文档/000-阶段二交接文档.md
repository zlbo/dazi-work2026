### 3.2 核心设计成果

#### 物理表设计（17张表）

**事实表（5张）**：
- fact_well_production - 井产量
- fact_well_injection - 井注入
- fact_stimulation - 措施作业
- fact_equipment_operation - 设备运行
- fact_pressure - 压力监测

**维度表（12张）**：
- dim_well, dim_block, dim_reservoir, dim_equipment, dim_date
- dim_stimulation_type, dim_well_type, dim_lifting_type
- dim_oil_gas_field, dim_formation, dim_equipment_type, dim_allocation

#### Cube设计（5个Cube）

| Cube名称 | 平台分类 | 核心度量 |
|----------|---------|---------|
| WellProductionCube | 流程型 | 日产油量、含水率、泵效 |
| BlockProductionCube | 主体型 | 月度产油量、产油率 |
| WellInjectionCube | 流程型 | 日注水量、注入压力 |
| EquipmentOperationCube | 快照型 | 运行时率、故障次数 |
| StimulationEffectCube | 流程型 | 增油量、有效率 |

#### Object Type设计（64个）

**核心Object Type**：
- Well（井）- 核心实体，贯穿7个业务域
- Reservoir（油藏）、Block（区块）、OilGasField（油气田）
- WellProduction（产量记录）、WellInjection（注水记录）
- Equipment（设备）、Stimulation（措施作业）
- PressureData（压力数据）、AlertRecord（预警记录）

**10域覆盖**：
- D1-D10全部覆盖，以Well为核心构建本体网络

#### Link Type设计（75个）

**关系分类**：
- 归属关系（15个）：belongsToReservoir、belongsToBlock、belongsToField
- 包含关系（7个）：containsBlock、containsWellbore
- 配套关系（6个）：equippedWithLift、equippedWithPump
- 生产关系（5个）：producesProduction、injectsIntoReservoir
- 作业关系（15个）：operatedByFracturing、operatedByAcidizing
- 评估监测关系（6个）：evaluatedByIntegrity、monitoredByPressure
- 关联关系（5个）：relatesToStation

#### Action Type设计（21个）

| 业务场景 | Action Type |
|---------|-------------|
| 生产管理 | AdjustLifting、ChangeStatus、PlanStimulation、RecordProduction、RecordInjection、MonitorProduction |
| 措施作业 | ExecuteFracturing、ExecuteAcidizing、ExecuteWorkover、ExecutePerforation、ExecuteProfileControl |
| 设备管理 | ScheduleMaintenance、RecordMaintenance、RecordFailure、MonitorEquipment |
| 安全环保 | ExecuteSafetyCheck、ExecuteEnvMonitoring、RecordIncident |
| 开发规划 | CreateDevelopmentPlan、CreateInjectionAllocation、CalculateProductionForecast |

#### 业务规则设计（95条）

| 规则类型 | 数量 | 编码范围 |
|---------|------|---------|
| 业务运算规则 | 25条 | BC001-BC025 |
| 业务逻辑规则 | 20条 | BL001-BL020 |
| 油藏工程规则 | 15条 | RE001-RE015 |
| 指标计算规则 | 20条 | IC001-IC020 |
| 预警规则 | 15条 | AR001-AR015 |

**核心规则示例**：
- BC001：日产油量 = 日产液量 × (1 - 含水率/100) × 原油密度
- BC014：注采比 = 月度注水量 / 月度产液量（目标范围0.8-1.5）
- AR001：产量异常预警（当前产量 < 平均值 × 70%）

## 五、关键设计约束

### 5.1 编码规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 物理表 | fact_*/dim_* + PascalCase | fact_well_production, dim_well |
