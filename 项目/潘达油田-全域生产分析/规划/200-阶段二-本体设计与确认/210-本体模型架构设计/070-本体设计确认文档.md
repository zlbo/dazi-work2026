### 3.2 数据流向确认

#### 数据架构层次

```
┌─────────────────────────────────────────────────────────────┐
│                    本体层（Ontology）                        │
│  Object Type: Well, Reservoir, Block, Equipment...         │
│  Link Type: belongsTo, contains, produces...               │
│  Action Type: AdjustLifting, ExecuteFracturing...          │
├─────────────────────────────────────────────────────────────┤
│                    Cube层（分析层）                          │
│  WellProductionCube, BlockProductionCube...                │
│  维度：date, well, block, reservoir...                      │
│  度量：daily_oil_production, avg_water_cut...              │
├─────────────────────────────────────────────────────────────┤
│                    物理表层（Table）                         │
│  fact_well_production, fact_well_injection...              │
│  dim_well, dim_block, dim_reservoir...                     │
└─────────────────────────────────────────────────────────────┘
```
