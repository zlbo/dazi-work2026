## 二、核心业务规则详细设计

### 2.1 业务运算规则（BC001-BC025）

#### BC001 日产油量计算规则

| 属性 | 内容 |
|------|------|
| **规则名称** | 日产油量计算规则 |
| **规则编码** | BC001 |
| **规则类型** | 业务运算规则 |
| **适用场景** | 单井日产量计算 |

**输入参数**：

| 参数名 | 数据类型 | 来源表 | 说明 |
|--------|---------|-------|------|
| daily_liquid_production | decimal(10,2) | fact_well_production | 日产液量(m³) |
| water_cut | decimal(5,2) | fact_well_production | 含水率(%) |

**计算公式**：
```
日产油量(t) = 日产液量(m³) × (1 - 含水率/100) × 原油密度(t/m³)
```

**输出结果**：

| 参数名 | 数据类型 | 说明 |
|--------|---------|------|
| daily_oil_production | decimal(10,2) | 日产油量(t) |

---

#### BC003 含水率计算规则

| 属性 | 内容 |
|------|------|
| **规则名称** | 含水率计算规则 |
| **规则编码** | BC003 |
| **规则类型** | 业务运算规则 |
| **适用场景** | 单井含水率计算 |

**输入参数**：

| 参数名 | 数据类型 | 来源表 | 说明 |
|--------|---------|-------|------|
| daily_water_production | decimal(10,2) | fact_well_production | 日产水量(m³) |
| daily_liquid_production | decimal(10,2) | fact_well_production | 日产液量(m³) |

**计算公式**：
```
含水率(%) = (日产水量 / 日产液量) × 100
```

**约束条件**：
- 日产液量 ≠ 0
- 0 ≤ 含水率 ≤ 100

### 2.2 业务逻辑规则（BL001-BL020）

#### BL001 压裂选井规则

| 属性 | 内容 |
|------|------|
| **规则名称** | 压裂选井规则 |
| **规则编码** | BL001 |
| **规则类型** | 业务逻辑规则 |
| **适用场景** | 筛选适合压裂的井 |

**输入参数**：

| 参数名 | 数据类型 | 来源表 | 说明 |
|--------|---------|-------|------|
| permeability | decimal(8,3) | dim_reservoir | 渗透率(mD) |
| decline_rate | decimal(5,2) | fact_well_production | 产量递减率(%/月) |
| water_cut | decimal(5,2) | fact_well_production | 含水率(%) |
| well_age | int | dim_well | 井龄(年) |

**判断条件**：

| 条件 | 阈值 | 说明 |
|------|------|------|
| 渗透率 | < 50 mD | 低渗储层 |
| 递减率 | > 5%/月 | 产量递减快 |
| 含水率 | < 80% | 含水较低 |
| 井龄 | < 10年 | 井龄适中 |

**决策结果**：
- ✅ 满足全部条件：推荐压裂
- ⚠️ 满足2-3项条件：可考虑压裂
- ❌ 满足少于2项：不推荐压裂

---

#### BL003 修井选井规则

| 属性 | 内容 |
|------|------|
| **规则名称** | 修井选井规则 |
| **规则编码** | BL003 |
| **规则类型** | 业务逻辑规则 |
| **适用场景** | 筛选需要修井的井 |

**输入参数**：

| 参数名 | 数据类型 | 来源表 | 说明 |
|--------|---------|-------|------|
| pump_efficiency | decimal(5,2) | fact_well_production | 泵效(%) |
| pump_cycle_days | int | - | 检泵周期(天) |
| current_cycle_days | int | - | 当前运行天数(天) |
| production_decline | decimal(5,2) | - | 产量下降幅度(%) |

**判断条件**：

| 条件 | 阈值 |
|------|------|
| 泵效 | < 50% |
| 运行天数 | > 检泵周期 × 1.2 |
| 产量下降 | > 30% |

**决策结果**：
- 满足任意2项：需要修井

### 2.3 油藏工程规则（RE001-RE015）

#### RE001 地质储量计算规则

| 属性 | 内容 |
|------|------|
| **规则名称** | 地质储量计算规则 |
| **规则编码** | RE001 |
| **规则类型** | 油藏工程规则 |
| **适用场景** | 油藏地质储量计算 |

**输入参数**：

| 参数名 | 数据类型 | 来源表 | 说明 |
|--------|---------|-------|------|
| area | decimal(10,4) | dim_block | 含油面积(km²) |
| thickness | decimal(6,2) | dim_reservoir | 有效厚度(m) |
| porosity | decimal(5,3) | dim_reservoir | 孔隙度(%) |
| saturation | decimal(5,2) | - | 含油饱和度(%) |
| oil_density | decimal(5,3) | - | 原油密度(t/m³) |

**计算公式**：
```
地质储量(t) = 面积(km²) × 10^6 × 厚度(m) × 孔隙度 × 含油饱和度 × 原油密度
```
