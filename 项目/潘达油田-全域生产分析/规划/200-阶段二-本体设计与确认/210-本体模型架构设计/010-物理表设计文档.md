> **版本**：V1.0

> **阶段**：阶段二 - 本体设计与确认

## 一、物理表设计概述

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| **星型模型** | 采用事实表-维度表的星型架构设计 |
| **领域驱动** | 基于10域业务模型进行表设计 |
| **核心实体优先** | 以"井"(Well)为核心设计表关系 |
| **编码规范** | 遵循`fact_*`、`dim_*`命名规范 |

### 1.2 表分类体系

| 表类型 | 命名规则 | 说明 | 数量 |
|-------|---------|------|------|
| **事实表** | `fact_*` | 记录业务事件和度量数据 | 5个 |
| **维度表** | `dim_*` | 描述业务实体属性 | 12个 |

## 二、核心事实表设计

### 2.1 fact_well_production（井产量事实表）

| 字段名 | 数据类型 | 长度 | 约束 | 说明 | 来源实体 |
|--------|---------|------|------|------|---------|
| production_key | string | 64 | PRIMARY KEY | 产量记录主键 | WellProduction |
| well_key | string | 64 | NOT NULL, FK→dim_well | 井主键 | Well |
| date_key | int32 | - | NOT NULL, FK→dim_date | 日期键 | - |
| daily_oil_production | decimal | (10,2) | - | 日产油量(t) | WellProduction |
| daily_liquid_production | decimal | (10,2) | - | 日产液量(m³) | WellProduction |
| daily_gas_production | decimal | (12,2) | - | 日产气量(m³) | WellProduction |
| water_cut | decimal | (5,2) | CHECK(0≤x≤100) | 含水率(%) | WaterCut |
| pump_efficiency | decimal | (5,2) | CHECK(0≤x≤100) | 泵效(%) | PumpEfficiency |
| working_hours | decimal | (5,2) | - | 工作小时(h) | ProductionStatus |
| status_code | string | 20 | - | 生产状态编码 | ProductionStatus |

| injection_temperature | decimal | (5,1) | - | 注入温度(℃) | - |
