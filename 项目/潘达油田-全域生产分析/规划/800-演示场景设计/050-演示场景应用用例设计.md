## 二、AI对话交互用例

### 2.1 用例清单

| 用例编号 | 用例名称 | 用户角色 | 业务场景 |
|---------|---------|---------|---------|
| UC-AI-001 | 产量查询 | 采油工程师 | 询问单井日产量 |
| UC-AI-002 | 趋势分析 | 生产主管 | 分析产量变化趋势 |
| UC-AI-003 | 异常预警 | 值班人员 | 了解异常事件 |
| UC-AI-004 | 效益查询 | 财务人员 | 查询单井效益 |
| UC-AI-005 | 设备状态 | 设备工程师 | 查询设备状态 |
| UC-AI-006 | 措施效果 | 工程师 | 评估措施效果 |
| UC-AI-007 | 根因分析 | 技术专家 | 分析产量下降原因 |
| UC-AI-008 | 智能建议 | 管理人员 | 获取优化建议 |

### 2.2 详细用例描述

#### UC-AI-001：产量查询

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井今天的产量是多少？" |
| **预期回答** | "潘达1井今日产量：原油25.3吨，液体68.5立方米，含水率63.1%。较昨日下降2.1%，主要受含水率上升影响。" |
| **数据来源** | fact_well_production |
| **展示形式** | 数值卡片 + 趋势指示 |

#### UC-AI-002：趋势分析

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井近30天产量变化趋势如何？" |
| **预期回答** | "潘达1井近30天产量呈缓慢下降趋势，日均产量从28吨降至25吨，降幅约10.7%。主要原因是含水率从58%上升至63%。建议关注注水效果。" |
| **数据来源** | fact_well_production |
| **展示形式** | 折线图 + 分析文字 |

#### UC-AI-003：异常预警

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井最近有什么异常情况？" |
| **预期回答** | "【预警提醒】潘达1井于2026-03-20出现含水率突升异常，含水率从60%升至72%，已持续5天。初步判断可能受邻井注水影响，建议调整注采参数。" |
| **数据来源** | fact_alert_record |
| **展示形式** | 预警卡片 + 处理建议 |

#### UC-AI-004：效益查询

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井本月效益如何？" |
| **预期回答** | "潘达1井本月预计收入：666.3万元，预计成本：4.57万元，预计利润：661.7万元。投入产出比为5.16，处于较好水平。" |
| **数据来源** | fact_well_production + dim_price |
| **展示形式** | 效益指标卡片 |

#### UC-AI-005：设备状态

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井的抽油机运行状态怎么样？" |
| **预期回答** | "潘达1井抽油机（型号CYJ10-3-53HB）当前运行正常。累计运行时间：52,360小时，下次计划维护：2026-10-25。近30天运行效率：89.2%。" |
| **数据来源** | dim_equipment + fact_maintenance_record |
| **展示形式** | 设备状态卡片 |

#### UC-AI-006：措施效果

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井上次压裂效果如何？" |
| **预期回答** | "潘达1井于2025-07-15实施酸化措施，投入费用8万元。措施后增油率23%，有效期105天，累计增油680吨，投入产出比4.3。效果良好。" |
| **数据来源** | fact_stimulation + fact_well_production |
| **展示形式** | 措施效果对比图 |

#### UC-AI-007：根因分析

| 项目 | 内容 |
|------|------|
| **用户问题** | "潘达1井产量为什么下降了？" |
| **预期回答** | "【根因分析】潘达1井产量下降主要原因：1）含水率上升（58%→63%），导致液油比降低；2）泵效略有下降（82%→78%）；3）受邻井W-PD-005注水量增加影响。建议：1）优化注水方案；2）安排泵况测试；3）与邻井协调注采参数。" |
| **数据来源** | 多表关联分析 |
| **展示形式** | 分析报告 + 建议 |

#### UC-AI-008：智能建议

| 项目 | 内容 |
|------|------|
| **用户问题** | "如何提高潘达1井的效益？" |
| **预期回答** | "【优化建议】针对潘达1井当前状况，建议：1）实施调剖措施，控制含水率上升；2）优化抽油机工作参数，提高泵效至85%以上；3）根据注采比分析，适当降低注水量。预计实施后单井日利润可提升15-20%。" |
| **数据来源** | 规则引擎 + 历史数据 |
| **展示形式** | 建议清单 + 预期效果 |

## 三、驾驶舱看板用例

### 3.1 看板布局设计

```
┌─────────────────────────────────────────────────────────────┐
│                    单井监控驾驶舱                           │

├───────────────┬───────────────┬───────────────┬───────────┤
│  产量指标     │  效益指标     │  设备指标     │  预警提示  │
│  • 日产油     │  • 日收入     │  • 运行状态   │  • 预警数  │
│  • 日产液     │  • 日利润     │  • 运行效率   │  • 处理状态 │
│  • 含水率     │  • 投入产出比 │  • 维护倒计时 │           │
├───────────────┴───────────────┴───────────────┴───────────┤
│  主图表区（Tab切换）                                        │
```

### 3.2 看板用例清单

| 用例编号 | 用例名称 | 功能描述 |
|---------|---------|---------|
| UC-DB-001 | 实时产量监控 | 展示当日产量、累计产量、趋势对比 |
| UC-DB-002 | 效益实时看板 | 展示收入、成本、利润实时数据 |
| UC-DB-003 | 设备健康监控 | 展示设备状态、运行参数、维护计划 |
| UC-DB-004 | 预警集中展示 | 展示未处理异常、预警等级分布 |
| UC-DB-005 | 趋势分析图表 | 产量/含水/泵效趋势对比 |
| UC-DB-006 | 措施效果评估 | 历史措施效果对比分析 |
| UC-DB-007 | 数据导出 | 导出选定时间段数据 |
| UC-DB-008 | 报告生成 | 一键生成单井分析报告 |

### 3.3 关键指标展示设计

| 指标类别 | 指标名称 | 展示形式 | 颜色规则 |
|---------|---------|---------|---------|
| 产量 | 日产油(t) | 大数字卡片 | 正常：绿色，下降>10%：黄色 |
| 产量 | 含水率(%) | 进度条 | <50%：绿色，50-80%：黄色，>80%：红色 |
| 效率 | 泵效(%) | 环形图 | >80%：绿色，60-80%：黄色，<60%：红色 |
| 效益 | 日利润(万元) | 数字卡片 | 盈利：绿色，亏损：红色 |
| 设备 | 运行状态 | 状态灯 | 运行：绿色，维护：黄色，故障：红色 |
| 预警 | 未处理数量 | 徽章 | 0：绿色，1-3：黄色，>3：红色 |

## 四、沙盘推演用例

### 4.1 推演场景设计

| 推演场景 | 业务目的 | 推演内容 |
|---------|---------|---------|
| 产量预测 | 预测未来产量变化 | 基于当前趋势预测3个月产量 |
| 措施对比 | 评估不同措施方案 | 压裂vs酸化效果对比 |
| 敏感性分析 | 分析参数影响 | 油价/成本变化对效益的影响 |
| 注采优化 | 优化注采参数 | 注水量调整方案评估 |
| 设备更新 | 评估设备投资 | 抽油机更新投资回报分析 |

### 4.2 推演用例清单

| 用例编号 | 用例名称 | 推演目标 | 输入参数 | 输出结果 |
|---------|---------|---------|---------|---------|
| UC-SD-001 | 产量递减预测 | 预测未来3个月产量 | 递减率、含水上升率 | 产量曲线、累计产量 |
| UC-SD-002 | 措施方案对比 | 对比不同措施效果 | 措施类型、费用、有效期 | 投入产出比、增油量 |
| UC-SD-003 | 油价敏感性分析 | 分析油价影响 | 油价波动范围 | 利润变化曲线 |
| UC-SD-004 | 注采参数优化 | 优化注水量 | 注采比范围 | 最优注水量建议 |
| UC-SD-005 | 设备投资评估 | 评估设备更新 | 设备成本、效率提升 | 投资回收期 |

### 4.3 详细推演用例

#### UC-SD-001：产量递减预测

| 项目 | 内容 |
|------|------|
| **推演目标** | 预测潘达1井未来3个月产量变化 |
| **输入参数** | 自然递减率：5%/月，含水上升率：1%/月 |
| **推演过程** | 基于历史数据拟合递减曲线，预测未来趋势 |
| **输出结果** | 预测曲线、累计产量、关键节点预警 |
| **决策建议** | 根据预测结果建议措施时机 |

#### UC-SD-002：措施方案对比

| 项目 | 内容 |
|------|------|
| **推演目标** | 对比压裂和酸化两种措施方案 |
| **输入参数** | 压裂费用：15万元，增油率30%，有效期120天 |
| | 酸化费用：8万元，增油率20%，有效期90天 |
| **推演过程** | 计算投入产出比、净收益、回收期 |
| **输出结果** | 方案对比表、效益对比图 |
| **决策建议** | 推荐投入产出比较高的方案 |

#### UC-SD-003：油价敏感性分析

| 项目 | 内容 |
|------|------|
| **推演目标** | 分析油价波动对单井效益的影响 |
| **输入参数** | 油价波动范围：±20% |
| **推演过程** | 计算不同油价下的日利润变化 |
| **输出结果** | 敏感性分析曲线、盈亏平衡点 |
| **决策建议** | 给出成本控制建议 |

## 五、用例优先级与实施建议

### 5.1 用例优先级

| 优先级 | 用例编号 | 用例名称 | 理由 |
|-------|---------|---------|------|
| P0 | UC-AI-001 | 产量查询 | 最基础最常用功能 |
