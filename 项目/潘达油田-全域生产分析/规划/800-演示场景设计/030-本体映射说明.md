## 三、Link Type映射

### 3.1 核心实体关联

| 关联关系 | Link Type | 说明 |
|---------|----------|------|
| Well → WellProduction | produces | 井产生产量记录 |
| Well → Equipment | hasEquipment | 井拥有设备 |
| Well → Stimulation | undergoes | 井经历措施作业 |
| Well → WellInjection | injectsTo | 注水井关联（注采关系） |
| Well → WellIntegrity | hasIntegrity | 井具有完整性状态 |
| Equipment → MaintenanceRecord | hasMaintenance | 设备有维护记录 |
| Equipment → FailureRecord | hasFailure | 设备有故障记录 |

### 3.2 跨域关联路径

```
Well (核心)
  ├─ produces → WellProduction (D4)
  ├─ hasEquipment → Equipment (D9)
  │     ├─ hasMaintenance → MaintenanceRecord
  │     └─ hasFailure → FailureRecord
  ├─ undergoes → Stimulation (D5)
  ├─ belongsTo → Block (D7)
  │     └─ contains → Reservoir (D1)
  └─ hasIntegrity → WellIntegrity (D10)
```

## 五、派生指标映射

### 5.1 收入指标

| 派生指标 | 计算逻辑 | 依赖对象 |
|---------|---------|---------|
| 日油收入 | WellProduction.daily_oil_production × Price.oil_price | WellProduction, Price |
| 日气收入 | WellProduction.daily_gas_production × Price.gas_price | WellProduction, Price |
| 日总收入 | 日油收入 + 日气收入 | 派生计算 |

### 5.2 成本指标

| 派生指标 | 计算逻辑 | 依赖对象 |
|---------|---------|---------|
| 措施日分摊 | Stimulation.cost_amount / Stimulation.valid_days | Stimulation |
| 日总成本 | WellCost.electricity_cost + WellCost.maintenance_cost + 措施日分摊 | WellCost, Stimulation |

### 5.3 效益指标

| 派生指标 | 计算逻辑 | 依赖对象 |
|---------|---------|---------|
| 日利润 | 日总收入 - 日总成本 | 派生计算 |
| 投入产出比 | (增油量 × 油价) / 措施成本 | Stimulation, Price |

## 七、实施建议

### 7.1 优先建模对象

| 优先级 | Object Type | 原因 |
|-------|------------|------|
| P0 | Well | 核心实体，必须优先 |
| P0 | WellProduction | 产量数据是基础 |
| P1 | Equipment | 设备健康视角必需 |
| P1 | Stimulation | 效益分析必需 |
| P2 | AlertRecord | 异常预警功能 |
| P2 | MaintenanceRecord | 维护记录功能 |

### 7.2 数据准备顺序

1. **基础维度**：Well → Block → Reservoir（建立主数据）
2. **核心事实**：WellProduction（生产数据）
3. **关联数据**：Equipment → MaintenanceRecord → FailureRecord（设备数据）
4. **措施数据**：Stimulation（措施作业数据）
5. **财务数据**：Price → WellCost（价格与成本数据）
