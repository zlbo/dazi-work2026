## 一、设计目标与原则

### 1.1 设计目标

| 目标 | 说明 |
|------|------|
| **高仿真度** | 数据分布、取值范围、变化趋势符合油田实际业务 |
| **强勾稽性** | 数据间满足严格的业务勾稽关系 |
| **可验证性** | 数据正确性可通过规则验证 |
| **演示友好** | 数据适合可视化展示，包含典型场景案例 |

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **业务真实性** | 所有数据遵循油田生产实际规律 |
| **勾稽完整性** | 液油平衡、注采平衡、能量平衡 |

## 四、数据验证规则

### 4.1 单表验证规则

```sql
-- 日产油量范围验证
SELECT * FROM fact_well_production 
WHERE well_key = 'W-PD-001'
  AND (daily_oil_production < 0 OR daily_oil_production > 50);

-- 含水率范围验证
SELECT * FROM fact_well_production 
WHERE well_key = 'W-PD-001'
  AND (water_cut < 0 OR water_cut > 100);

-- 液油勾稽验证
SELECT * FROM fact_well_production 
WHERE well_key = 'W-PD-001'
  AND ABS(daily_liquid_production * (1 - water_cut/100.0) - daily_oil_production) > 0.5;
```

### 4.2 跨表验证规则

```sql
-- 设备与井关联验证
SELECT * FROM dim_equipment
WHERE well_key = 'W-PD-001'
  AND status_code NOT IN ('RUNNING', 'MAINTENANCE');

-- 措施与产量关联验证
WITH stim_effect AS (
    SELECT 
        s.well_key,
        s.date_key as stim_date,
        s.valid_days,
        s.result_oil_increase
    FROM fact_stimulation s
    WHERE s.well_key = 'W-PD-001'
)
SELECT 
    p.date_key,
    p.daily_oil_production,
    AVG(p.daily_oil_production) OVER (ORDER BY p.date_key ROWS BETWEEN 30 PRECEDING AND 1 PRECEDING) as avg_before
FROM fact_well_production p
JOIN stim_effect se ON p.well_key = se.well_key
WHERE p.date_key BETWEEN se.stim_date AND se.stim_date + se.valid_days
  AND p.daily_oil_production < avg_before * 1.1; -- 措施后应增产
```
