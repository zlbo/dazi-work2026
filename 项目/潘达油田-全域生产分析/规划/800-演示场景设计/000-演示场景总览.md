## 一、场景定位

### 1.1 场景名称
**单井生产监控与效益分析**

### 1.2 核心目标
为油田生产管理人员提供**单一井的全景视图**，涵盖生产运营、设备健康、财务效益三大维度，支持数据驱动的决策分析。

### 1.3 设计原则

| 原则 | 说明 |
|------|------|
| **核心实体聚焦** | 以"井"(Well)为核心，贯穿全业务链 |
| **多视角融合** | 生产、设备、效益三位一体 |
| **可视化优先** | 图表化展示关键指标趋势 |
| **预警联动** | 异常事件自动触发预警提示 |

### 1.4 目标用户

| 用户角色 | 关注点 |
|---------|--------|
| 采油工程师 | 产量变化、生产异常、措施效果 |
| 设备工程师 | 设备状态、维护计划、故障预警 |
| 生产管理人员 | 综合效益、成本控制、绩效评估 |
| 财务人员 | 成本核算、投入产出比 |

## 二、场景架构

### 2.1 三大视角架构

```
┌─────────────────────────────────────────────────────────────┐
│                    单井全景视图                             │
├─────────────────────────────────────────────────────────────┤
│  井基本信息栏（井号、井型、区块、油藏、举升方式、开钻日期）    │
├─────────────────────┬─────────────────────┬───────────────┤
│   生产运营视角      │   设备健康视角      │   财务效益视角  │
│   (Production)     │   (Equipment)       │   (Finance)    │
├─────────────────────┼─────────────────────┼───────────────┤
│ • 日产量曲线       │ • 设备状态概览      │ • 日收入/利润   │
│ • 含水率分析       │ • 维护记录          │ • 成本构成      │
│ • 举升效率         │ • 故障预警          │ • 投入产出比    │
│ • 生产异常事件     │ • 寿命预测          │ • 效益指标      │
└─────────────────────┴─────────────────────┴───────────────┘
```

### 2.2 跨域关联关系

| 关联方向 | 业务关系 | 涉及Object |
|---------|---------|-----------|
| Well → WellProduction | 生产记录 | Well, WellProduction |
| Well → Equipment | 设备归属 | Well, Equipment |
| Well → Stimulation | 措施历史 | Well, Stimulation |
| Well → WellInjection | 注采关联 | Well, WellInjection |
| Well → WellIntegrity | 安全监控 | Well, WellIntegrity |

## 三、核心功能模块

### 3.1 生产运营视角

| 功能模块 | 说明 | 数据来源 |
|---------|------|---------|
| 产量监控 | 日油/液/气产量趋势 | fact_well_production |
| 含水率分析 | 含水率变化曲线 | fact_well_production |
| 举升效率 | 泵效监控与趋势 | fact_well_production |
| 生产异常 | 异常事件记录与预警 | AlertRecord |

### 3.2 设备健康视角

| 功能模块 | 说明 | 数据来源 |
|---------|------|---------|
| 设备概览 | 关联设备清单与状态 | Equipment |
| 维护记录 | 维护历史与计划 | MaintenanceRecord |
| 故障预警 | 设备故障预测与预警 | FailureRecord |
| 寿命预测 | 设备剩余寿命评估 | Equipment |

### 3.3 财务效益视角（新增）

| 功能模块 | 说明 | 数据来源 |
|---------|------|---------|
| 收入计算 | 基于产量的收入估算 | fact_well_production + 价格参数 |
| 成本分析 | 操作成本、维护成本构成 | 成本维度表 |
| 投入产出 | 措施投入与增油收益对比 | Stimulation |
| 效益指标 | 综合效益评估指标 | 派生计算 |
