## 一、量化规则定义

### 1.1 量化规则概念

| 定义项 | 说明 |
|--------|------|
| **量化规则** | 可用数学公式或逻辑表达式描述的业务规则 |
| **规则类型** | 运算规则、逻辑规则、工程规则、指标规则、预警规则 |
| **规则编码** | 统一的规则标识编码 |
| **规则参数** | 规则计算所需的输入参数 |
| **规则输出** | 规则计算产生的输出结果 |
