## 一、流程与动作定义

### 1.1 业务流程定义

| 定义项 | 说明 |
|--------|------|
| **业务流程** | 完成某项业务目标的一系列有序动作 |
| **流程节点** | 流程中的关键步骤，对应一个Action Type |
| **触发条件** | 启动流程的前提条件 |
| **执行主体** | 流程的执行者或执行系统 |
| **输出结果** | 流程完成后的产出 |

### 1.2 Action Type定义

| 定义项 | 说明 | 示例 |
|--------|------|------|
| **Action Type代码** | 无前缀PascalCase | OpenWell |
| **显示名称** | 中文显示名 | 开井 |
| **触发条件** | 动作启动的前提 | 井状态为关井 |
| **执行主体** | 动作的执行者 | 采油工/调度员 |
| **输入参数** | 动作需要的输入 | 井号、开井时间 |
| **输出结果** | 动作的产出 | 生产状态更新 |
