## 一、维度定义与分类

### 1.1 维度定义

| 定义项 | 说明 |
|--------|------|
| **维度** | 用于数据分类、筛选、聚合的标准属性或编码 |
| **维度表** | 包含维度编码、维度名称、描述等信息的标准化表格 |
| **编码规则** | 维度值的编码规范，确保唯一性和可读性 |
| **层级关系** | 维度之间的父子层级关系（如区块→井组→单井） |

### 1.2 维度分类

| 维度分类 | 英文名称 | 维度数量 | 说明 |
|---------|---------|---------|------|
| **井相关维度** | WellDimension | 8 | 井型、井别、井状态等 |
| **设备相关维度** | EquipmentDimension | 5 | 设备类型、设备状态等 |
| **区块相关维度** | BlockDimension | 3 | 区块编码、开发阶段等 |
| **生产相关维度** | ProductionDimension | 4 | 生产状态、措施类型等 |
| **安全环保维度** | SafetyDimension | 3 | 隐患等级、风险等级等 |

**维度表总数**：23个

## 七、维度层级关系

### 7.1 空间层级关系

```
油田 (Oilfield)
└── 区块 (Block)
    └── 井组 (WellGroup)
        └── 单井 (Well)
            └── 井段 (Interval)
```

### 7.2 设备层级关系

```
系统 (System)
└── 子系统 (SubSystem)
    └── 设备类型 (EquipmentType)
        └── 设备实例 (Equipment)
```

### 7.3 时间层级关系

```
年度 (Year)
└── 季度 (Quarter)
    └── 月度 (Month)
        └── 日 (Day)
            └── 班次 (Shift)
```
