## 一、隐性业务关系定义

### 1.1 隐性关系特征

| 特征 | 说明 | 示例 |
|------|------|------|
| **数据不直接体现** | 数据表中没有直接字段表示该关系 | 含水率上升→产量下降（需要计算分析才能发现） |
| **经验驱动** | 需要行业经验才能识别和利用 | 压裂后30天见效（需要历史数据验证） |
| **决策价值高** | 对生产决策有重要指导作用 | 含水率突破临界值→需要措施干预 |
| **可量化潜力** | 部分可以量化为公式或阈值 | 递减率公式、含水率预警阈值 |

### 1.2 隐性关系分类

| 分类 | 英文名称 | 业务含义 | 平台映射 |
|------|---------|---------|---------|
| **指标关联规律** | IndicatorCorrelation | 指标之间存在因果或相关关系 | 派生属性/计算规则 |
| **时间延迟效应** | TimeDelayEffect | 决策影响有延迟显现 | 预警规则/预测模型 |
| **临界阈值规律** | CriticalThreshold | 达到临界点触发状态变化 | 预警规则/派生属性 |
| **异常征兆规律** | AnomalySign | 早期征兆预示后续问题 | 预警规则 |
| **业务前提假设** | BusinessAssumption | 业务规则成立的隐含前提 | 业务逻辑规则 |
| **经验公式规律** | EmpiricalFormula | 行业经验形成的计算公式 | 派生属性 |

## 三、可量化隐性关系详细定义

### 3.1 指标关联规律（可量化）

| 关系编号 | 关系名称 | 量化公式 | 参数定义 | 阈值/临界值 |
|---------|---------|---------|---------|-----------|
| H001 | 含水率上升导致产量下降 | `日产油量 = 日产液量 × (1 - 含水率)` | 日产液量、含水率 | 含水率>80%触发预警 |
| H002 | 产量递减规律（Arps公式） | `q(t) = qi / (1 + b·Di·t)^1/b` | qi初始产量、Di初始递减率、b递减指数 | Di>10%/年需关注 |
| H006 | 日产油低于阈值触发低产预警 | `预警条件 = 日产油量 < 0.5t` | 日产油量 | 阈值0.5t/d |
| H009 | 注采比失衡导致压力下降 | `注采比 = 注水量 / 产液量` | 注水量、产液量 | 注采比<1触发预警 |
| H020 | 电潜泵排量与井产液量匹配 | `匹配率 = 实际排量 / 额定排量` | 实际排量、额定排量 | 匹配率应在70%-90% |
| H024 | 井深与举升方式选择 | `举升方式选择规则：井深>2500m优选ESP` | 井深 | 阈值2500m |
| H026 | 区块产量与区块井数正相关 | `区块产量 = Σ单井产量` | 单井产量、井数 | 无阈值 |
| H027 | 单井成本与产量负相关 | `吨油成本 = 单井总成本 / 日产油量` | 单井总成本、日产油量 | 吨油成本>阈值触发预警 |
| H032 | 注水压力超过破裂压力导致窜流 | `窜流风险 = 注入压力 > 破裂压力` | 注入压力、破裂压力 | 破裂压力为临界值 |
| H036 | 集输站处理能力与井产量匹配 | `匹配率 = 区域总产量 / 集输站处理能力` | 区域总产量、处理能力 | 匹配率应在80%-95% |
| H039 | 油藏采收率与开发方式相关 | `采收率 = 可采储量 / 地质储量` | 可采储量、地质储量 | 采收率目标值 |
| H040 | 井报废时机与经济效益 | `经济极限产量 = 固定成本 / (油价 - 变动成本)` | 固定成本、油价、变动成本 | 产量<经济极限产量 |
