## 三、油田行业Object Type设计

### 3.1 核心Object Type（10域覆盖）

```yaml
油田行业核心Object Type:
  # 地质勘探域
  - Object Type: Reservoir
    display_name: 油藏
    description: 地下油气的聚集单元
    attributes:
      - name: reservoir_code
        type: string
        description: 油藏编码
      - name: reservoir_name
        type: string
        description: 油藏名称
      - name: reservoir_type
        type: string
        description: 油藏类型（稀油/稠油）
      - name: porosity
        type: decimal
        description: 孔隙度
      - name: permeability
        type: decimal
        description: 渗透率(md)

  - Object Type: Formation
    display_name: 地层
    description: 地质层位
    attributes:
      - name: formation_code
        type: string
        description: 地层编码
      - name: formation_name
        type: string
        description: 地层名称
      - name: depth
        type: decimal
        description: 深度(m)

  - Object Type: Fluid
    display_name: 流体
    description: 油藏中的流体（油、气、水）
    attributes:
      - name: fluid_type
        type: string
        description: 流体类型（油/气/水）
      - name: viscosity
        type: decimal
        description: 粘度(mPa·s)
      - name: density
        type: decimal
        description: 密度(kg/m³)

  # 钻井工程域
  - Object Type: Wellbore
    display_name: 井筒
    description: 井眼的工程结构
    attributes:
      - name: wellbore_code
        type: string
        description: 井筒编码
      - name: total_depth
        type: decimal
        description: 井深(m)
      - name: casing_program
        type: string
        description: 套管程序

  - Object Type: DrillingRig
    display_name: 钻井设备
    description: 钻井作业设备
    attributes:
      - name: rig_code
        type: string
        description: 钻机编号
      - name: rig_type
        type: string
        description: 钻机类型
      - name: capacity
        type: decimal
        description: 钻机能力(t)

  # 采油采气生产域（核心）
  - Object Type: Well
    display_name: 井
    description: 油田核心实体，贯穿钻井、完井、生产全过程
    attributes:
      - name: well_code
        type: string
        description: 井号
      - name: well_name
        type: string
        description: 井名称
      - name: well_type
        type: string
        description: 井型（采油井/注水井/观察井）
      - name: lifting_type
        type: string
        description: 举升方式
      - name: status
        type: string
        description: 井状态
      - name: spud_date
        type: date
        description: 开钻日期
      - name: completed_date
        type: date
        description: 完井日期
    relationships:
      - link: locatedIn
        target: Block
        description: 所属区块
      - link: hasReservoir
        target: Reservoir
        description: 所属油藏
      - link: hasFormation
        target: Formation
        description: 穿遇地层

  - Object Type: WellProduction
    display_name: 井产量
    description: 井的日常产量数据
    attributes:
      - name: production_date
        type: date
        description: 生产日期
      - name: daily_oil_production
        type: decimal
        description: 日产油量(t/d)
      - name: daily_liquid_production
        type: decimal
        description: 日产液量(m³/d)
      - name: daily_gas_production
        type: decimal
        description: 日产气量(m³/d)
      - name: water_cut
        type: decimal
        description: 含水率(%)
      - name: pump_efficiency
        type: decimal
        description: 泵效(%)

  - Object Type: WellInjection
    display_name: 井注入
    description: 注水井的注入数据
    attributes:
      - name: injection_date
        type: date
        description: 注入日期
      - name: daily_water_injection
        type: decimal
        description: 日注水量(m³/d)
      - name: injection_pressure
        type: decimal
        description: 注入压力(MPa)

  # 地面集输域
  - Object Type: GatheringStation
    display_name: 集输站
    description: 油气集输站场
    attributes:
      - name: station_code
        type: string
        description: 站场编码
      - name: station_name
        type: string
        description: 站场名称
      - name: station_type
        type: string
        description: 站场类型

  - Object Type: Pipeline
    display_name: 管线
    description: 油气集输管线
    attributes:

        type: string
        description: 管线编号

        type: decimal
        description: 长度(m)
      - name: diameter
        type: decimal
        description: 直径(mm)

    display_name: 设备
    description: 油田生产设备

      - name: equipment_code
        type: string
```

## 四、油田行业Link Type设计

### 4.1 核心Link Type

```yaml
油田行业核心Link Type:
  # 空间关系
  - Link Type: locatedIn
    display_name: 位于
    description: 实体所在的空间位置关系
    source: Well | GatheringStation | Pipeline
    target: Block | Field

  - Link Type: belongsTo
    display_name: 属于
    description: 归属关系
    source: Wellbore | WellProduction | Equipment
    target: Well

  - Link Type: connectsTo
    display_name: 连接
    description: 管线连接关系
    source: Pipeline
    target: Well | GatheringStation

  # 生产关系
  - Link Type: produces
    display_name: 生产
    description: 井的生产关系
    source: WellProduction
    target: Well

  - Link Type: injects
    display_name: 注入
    description: 注水井的注入关系
    source: WellInjection
    target: Well

  - Link Type: hasReservoir
    display_name: 产于
    description: 井与油藏的归属关系
    source: Well
    target: Reservoir

  - Link Type: hasFormation
    display_name: 穿遇
    description: 井筒与地层的关系
    source: Wellbore
    target: Formation

  # 业务关系
  - Link Type: drilledBy
    display_name: 由...钻井
    description: 钻井作业关系
    source: Wellbore
    target: DrillingRig

  - Link Type: monitoredBy
    display_name: 受...监控
    description: 设备监控关系
    source: Equipment
    target: GatheringStation

  # 分析关系
  - Link Type: aggregatesTo
    display_name: 汇总至
    description: 数据汇总关系

    target: BlockProduction
```

## 八、业务规则定义

### 8.1 油田行业核心业务规则

```yaml
油田行业核心业务规则:
  # 产量计算规则
  - rule_name: 日产油量计算规则
    rule_code: PRD001
    formula: 日产油量 = 日产液量 × (1 - 含水率/100)
    parameters:
      - name: 日产液量
        source: fact_well_production.daily_liquid_production
      - name: 含水率
        source: fact_well_production.water_cut

  - rule_name: 注采比计算规则
    rule_code: INJ001
    formula: 注采比 = 月度注水量 / 月度产液量
    parameters:
      - name: 月度注水量
        source: fact_well_injection
      - name: 月度产液量
        source: fact_well_production

  # 约束规则
  - rule_name: 含水率范围约束
    rule_code: WAT001
    constraint: 0 ≤ 含水率 ≤ 100
    source: fact_well_production.water_cut

  - rule_name: 产量非负约束
    rule_code: PRD101
    constraint: 日产油量 ≥ 0
    source: fact_well_production.daily_oil_production

  # 勾稽规则
  - rule_name: 液量与油量勾稽
    rule_code: CR001
    formula: 日产液量 = 日产油量 / (1 - 含水率/100)
    tolerance: 0.5%
    source: fact_well_production
```
