### 1.1 业务背景

化工生产企业以**装置/工艺单元**为核心生产单元，原料投入、能源消耗与产品产出形成完整的成本链路。本方案构建**化工行业利润成本分析本体**，以业务语言描述「会计科目」「实际分录」「预算」「原料」「装置」「能耗」及其关系，支撑：

- **损益总览**：收入、成本、费用、营业利润、利润率（按期间/组织/装置）
- **成本结构**：按科目层级展开直接材料/直接人工/制造费用/能源成本
- **装置成本**：按装置/单元归集成本，与生产实绩联动
- **原料与能耗**：原料消耗、能耗成本（电力/蒸汽/燃料），化工特色分析
- **实际 vs 预算**：按期间、科目、成本中心、装置的预算执行率与差异分析
- **组织维度**：按成本中心/部门/装置区贡献与超支预警
- **趋势分析**：同比、环比利润与关键科目走势

### 1.2 核心业务问题

| 问题 | 说明 |
| ---- | ---- |
| 利润是多少 | 期间营业利润、利润率 |
| 钱从哪来/到哪去 | 收入、成本、费用科目结构 |
| 各装置/单元成本多少 | 按装置归集的成本、原料消耗、能耗 |
| 预算执行如何 | 预算额、实际额、差异额、执行率（按组织/装置） |
| 能耗占比多少 | 能源成本占总成本比例、单位产品能耗 |
| 哪些科目/装置异常 | Top N、同比/环比异动 |
| 趋势如何 | 按月/季/年利润与关键科目走势 |

### 1.3 与参照示例的差异说明

| 维度 | 利润示例 | 设备运营 | 本方案（化工利润成本） |
| ---- | -------- | -------- | ---------------------- |
| 核心锚点 | 科目/分录/预算 | 设备/装置/停机/维保 | **科目/装置/原料/能耗 + GL分录/预算** |
| 主事实粒度 | 凭证行 | 设备日运行/停机事件 | **GL分录行 + 生产成本归集 + 能耗记录** |
| 核心 KPI | 收入、成本、利润率 | OEE、可用率、MTBF | **利润、装置成本、能耗强度、原料单价** |
| 组织维度 | 成本中心 | 厂区→工艺单元→设备 | **成本中心 + 厂区→工艺单元（化工组织结构）** |
| 特色维度 | GL财务 | 化工设备 | **原料（化工原料）、能耗（电/蒸汽/燃料）、装置** |
| 预实分析 | 科目级预算vs实际 | 计划vs实际运行 | **科目+装置双维度预算执行** |
| 函数设计 | 科目/利润项结构 | OEE/停机/维保/能耗专项 | **损益总览/装置成本/能耗分析/预算执行** |

**维度**：entry_id, line_id, date_key, calendar_date, year, month, account_id, account_code, account_name, account_type, pl_category, cost_element, cost_center_id, cost_center_name, plant_id, unit_id, fiscal_year, fiscal_period

### 8.1 风险

| 等级 | 风险 | 缓解措施 |
| ---- | ---- | -------- |
| 中 | 原料/能源数据与GL分录口径不一致 | 规划统一 `cost_element` 分类；seed 时对齐 |

| 中 | 多成本中心共用装置成本分摊 | 规划说明分摊规则；函数按 cost_center_id 过滤 |
| 低 | 多域共用 space_id + dim_plant/unit | init ensure 不重复建表 |
| 低 | 预算期间粒度（月度 vs 年度） | 规划明确 fiscal_period 口径；函数支持 period 汇总 |
