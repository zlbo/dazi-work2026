### 1.1 业务背景

化工/流程制造企业的**生产计划域**以主生产计划（MPS）、工单排程与物料需求为核心，连接销售预测、装置产能与车间执行。本方案构建**生产计划分析本体**，以业务语言描述「工厂」「产线/工作中心」「产品」「计划版本」「生产计划行」「生产工单」「日生产实绩」「物料需求」及其关系，支撑：

- **计划总览**：计划产量、实际产量、计划达成率、在制工单数
- **计划对比**：MPS 计划 vs 工单下达 vs 日实绩（按产品/产线/工厂）
- **工单执行**：工单状态结构、按期完工率、延期工单聚焦
- **产能负荷**：可用工时 vs 计划负荷 vs 实际负荷、瓶颈产线识别
- **物料配套**：计划需求量 vs 实际发料量、缺口与齐套率
- **产品结构**：品种/大类产量占比、产线产品组合
- **趋势对标**：产量同比/环比、产线横向对标

### 1.2 核心业务问题

| 问题 | 说明 |
| ---- | ---- |
| 计划排多少 | MPS 按产品/产线/周期的计划产量与工时 |
| 实际做了多少 | 日实绩、合格产量、报废/返工 |
| 计划达成没有 | 计划达成率、按期完工率、偏差归因 |
| 工单跑完没有 | 工单状态、在制、延期、Top 延期工单 |
| 产能够不够用 | 工作中心负荷率、超负荷/闲置识别 |
| 料齐不齐 | 物料需求 vs 发料、缺口与齐套率 |
| 哪条产线是短板 | 产线对标、产品 mix 结构 |
| 趋势如何 | 产量/达成率同比、环比 |

### 1.3 与参照示例的差异说明

| 维度 | 销售示例 | 设备运营示例 | 利润示例 | 本方案（生产计划） |
| ---- | -------- | ------------ | -------- | ------------------ |
| 核心锚点 | 订单/客户/渠道 | 设备/OEE/停机/维保 | 科目/凭证/预算 | **MPS/工单/产线/物料需求** |
| 主事实粒度 | 销售订单行 | 设备日运行 + 停机事件 | 凭证行 | **计划行 + 工单 + 日生产实绩** |
| 核心 KPI | 销售额、渠道占比 | OEE、可用率、MTBF | 收入、成本、利润率 | **计划达成率、负荷率、齐套率、按期完工率** |
| 组织维度 | 区域/渠道 | 厂区 → 工艺单元 → 设备 | 成本中心 | **工厂 → 工作中心/产线** |
| 对比分析 | 同比销售 | 运行计划 vs 实际运行 | 预实对比（财务） | **MPS vs 工单 vs 日实绩、物料计划 vs 发料** |
| 时间语义 | 订单日期 | 统计日/班次 | 会计期间 | **计划版本（月/周）+ 工单计划日 + 实绩日** |
| 行业特征 | 通用零售 | 化工装置连续生产 | 制造财务 GL | **流程制造排产、批次/连续混合、MRP 配套** |
| 函数设计 | 销售结构/Top 产品 | OEE/停机/能耗 | 科目/利润结构 | **计划达成/产能负荷/物料缺口/工单延期** |

> **与设备运营的分工**：设备运营关注**单台设备**可用性与 OEE；本域关注**产线/工厂级**排产计划、工单执行与物料配套。二者可共用 `dim_plant`、`dim_date`，但 init **仅**创建/注册本域表；若维表已存在则 **ensure** 不重复建表。  
> **与销售的分工**：销售回答「卖了多少给谁」；本域回答「工厂计划生产多少、实际产多少、何时交付产能」。

## 八、风险与验收

### 8.1 风险

| 等级 | 风险 | 缓解措施 |
| ---- | ---- | -------- |
| 中 | MPS 与日实绩粒度不一致（月计划 vs 日实绩） | PlanVsActualCube 按 fiscal_month 聚合 daily；函数注释写明聚合规则 |
| 中 | 多班次同日多行重复计量 | daily_id 含 shift；汇总函数默认按 work_center+product+date 去重或 sum |
| 中 | 工单与计划行一对多 | wo_from_plan 允许空 plan_line_id；plan_vs_actual 以 daily 为主对比 |
| 低 | 多域共用 space_id | init 仅创建本域表；dim_date/plant ensure |
| 低 | 物料需求与 BOM 未建模 | 本方案以 fact_material_requirement 承载结果集，不建 BOM 桥表 |
