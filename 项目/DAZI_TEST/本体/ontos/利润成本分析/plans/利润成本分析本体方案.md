### 1.1 业务背景

利润成本分析是企业财务管理的核心，旨在对收入、成本、费用进行结构化分析，支撑：

- **利润结构分析**：按科目、利润项、成本项展开损益构成
- **成本分析**：成本项分类汇总、成本归集与分摊
- **预实对比**：预算与实际执行对比，差异归因
- **趋势分析**：同比、环比利润与关键科目走势
- **组织分析**：按成本中心/部门分析利润贡献

### 1.2 与利润示例的差异

| 维度 | 利润示例 | 本方案（利润成本分析） |
| ---- | -------- | ---------------------- |
| 核心焦点 | GL凭证/预算预实 | 利润项/成本项结构化分析 |
| 科目结构 | 通用科目树 | 按利润项/成本项分类展开 |
| 成本维度 | 成本中心 | 成本项（直接/间接）+ 成本中心 |
| 分析目标 | 利润中心视角 | 利润项/成本项视角 |
| 函数设计 | 通用GL函数 | 利润项/成本项专项分析 |

### 4.1 ActualCube（Process · 实际发生主 Cube）

**事实源**：`fact_gl_journal_entry`

**维度**：entry_id, line_id, date_key, posting_date, fiscal_year, fiscal_period, year, month, account_id, account_code, account_type, pl_category, profit_item_id, cost_item_id, cost_center_id, cost_center_name, department, voucher_no

**度量**：

| 度量名       | 列名          | 聚合 | 说明       |
| ------------ | ------------- | ---- | ---------- |
| debit_total  | debit_amount  | sum  | 借方合计   |
| credit_total | credit_amount | sum  | 贷方合计   |
| net_amount   | amount_signed | sum  | 损益符号额 |
| line_count   | line_id       | count| 分录行数  |

**派生度量**：

| 度量名           | 表达式（示意）                                  | 说明     |
| ---------------- | --------------------------------------------- | -------- |
| revenue          | sumIf(amount_signed, account_type='收入')     | 收入     |
| cost             | sumIf(amount_signed, account_type='成本')     | 营业成本 |
| expense          | sumIf(amount_signed, account_type='费用')     | 期间费用 |
| gross_profit     | revenue + cost                               | 毛利     |
| operating_profit | gross_profit - expense                        | 营业利润 |
| profit_margin    | if(revenue>0, operating_profit/revenue, 0)   | 利润率   |

## 八、风险与验收

### 8.1 风险

| 等级 | 风险                      | 缓解措施                                  |
| ---- | ------------------------- | ---------------------------------------- |
