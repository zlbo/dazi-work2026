# 快速启动

欢迎开始 **应用项目（DRAP monorepo）** 开发。

## 1. 项目结构

本目录即 DRAP monorepo 根（含 `sdk/`、`templates/`、`apps/`）。应用组件位于 `apps/<app_id>/`。

相对工作区：`项目/app_应用0519`

## 2. 本地开发（首个组件）

```bash
cd apps/alarm-center
pnpm run dev
```

若依赖未安装，请在**应用项目根目录**执行 `pnpm install`。

浏览器页模式：`http://localhost:3011/runtime-apps/alarm-center`

## 3. 数据空间

绑定空间：**数据空间0519**（`space__0519`）

## 4. 构建与发布

```bash
# 在应用组件目录
cd apps/<app_id>
pnpm run build
# 在应用项目根
pnpm run dazi-app -- upload --cwd apps/<app_id> --space space__0519 --activate
```

扩展内参考：`资源/docs/app/build-upload.md`（先 `dazi docs sync`）。

## 5. 下一步

1. 阅读 `规划/`、`AGENTS.md`
2. 在 `apps/` 下维护多个应用组件
3. 侧栏 **App 应用** 树中展开本项目进行 dev / upload
